# 视觉表达学习流程

用于学习优秀主图的视觉表达方法，重点记录“它为什么好看、为什么亮眼、为什么信息多但不乱、为什么特效不抢主体”。

这个流程适合在拆完基础卖点后使用。先看视觉，再抽象方法，最后转成可复用提示词和设计规则。

---

## 学习目标

视觉学习不是评价“好不好看”，而是拆出可复用的方法。

```text
产品为什么亮眼：
特效为什么不抢主体：
卖点卡为什么清晰：
背景为什么有设计感：
颜色为什么协调：
信息为什么多但不乱：
哪些方法可以迁移到下一个产品：
```

---

## 第一步：先判断主体是否成立

优秀主图的第一标准是产品主体可信、清楚、亮眼。

```text
产品是否是全图最大视觉主体：
产品是否完整露出：
产品边缘是否清楚：
Logo/面板/关键结构是否可读：
材质是否有高光和反光：
主体亮度是否高于背景和卖点卡：
是否存在特效、文字、赠品抢主体：
```

可复用判断：

```text
如果主体不够亮，优先增加产品边缘高光、材质反光、局部补光。
如果背景太抢，降低背景亮度和饱和度。
如果文字太抢，把文字放到后景、降低实心度或改为同色系光效。
```

---

## 第二步：拆特效字如何服务主卖点

特效字不是装饰，它要同时完成“抓眼”和“强化卖点”。

```text
特效字内容是否就是核心卖点：
特效字在产品前方还是后方：
是否遮挡产品关键位置：
字体颜色是否与产品高光同色系：
是否使用烟雾、火焰、光效、描边、半透明等方式融入背景：
特效字的亮度是否低于或接近产品主体：
```

好用结构：

```text
核心卖点大字放后景
同色系光效连接产品高光
字形可以大，但避开产品面板、Logo、主要轮廓
让字成为氛围，不成为另一个主体
```

---

## 第三步：拆卖点卡如何清楚但不抢主体

卖点卡负责解释，产品主体负责吸引和信任。

```text
卖点卡位置：
卖点卡底色：
卖点卡亮度：
文字对比度：
卖点数量：
编号/图标/标签如何分节奏：
是否有局部图或过程图支撑卖点：
是否比产品主体更抢眼：
```

推荐方法：

```text
卖点卡用黑、灰、深棕、米灰等稳定底色。
标题用白字或浅色字保证可读。
编号、箭头、功能标签用少量橙色/金色/红色点亮。
卖点卡亮度低于产品主体，但文字对比必须足够高。
卖点卡内部按 1-3 个证据点组织，不堆满长句。
```

---

## 第四步：拆背景如何有设计但不杂乱

背景的作用是承托产品和氛围，不是展示自己。

```text
背景主色：
背景是否有空间层次：
是否有光源方向：
是否有材质、星点、边框、光束、烟雾、舞台感：
背景亮度是否低于产品主体：
背景元素是否围绕产品轮廓服务：
背景是否影响文字可读性：
```

推荐方法：

```text
深色背景承托浅色产品。
暖色底光增强烹饪温度和促销气氛。
金属轮廓或光框增加科技感和新品感。
星点、烟雾、光效只做氛围，不做主体。
背景越丰富，越要控制亮度和饱和度。
```

---

## 第五步：拆色彩分工

优秀主图通常不是颜色越多越好，而是主色、辅色、点缀色分工明确。

```text
主色：
辅色：
点缀色：
产品色与背景色是否有反差：
特效字是否跟产品高光同色系：
功能标签是否使用统一点缀色：
是否存在无意义的多色混用：
```

常见分工：

| 色彩角色 | 用途 |
| --- | --- |
| 主色 | 决定整体调性，如黑、白、银、红 |
| 辅色 | 承接背景、卡片、材质层次 |
| 点缀色 | 强调编号、功能、价格、火力、权益 |

---

## 第六步：拆信息动线

信息动线决定用户先看什么、后看什么。

```text
第 1 眼：
第 2 眼：
第 3 眼：
第 4 眼：
最后促成点击的信息：
```

推荐动线：

```text
品牌/平台背书
-> 核心大卖点特效字
-> 产品主体
-> 左侧卖点卡
-> 局部证据/过程图
-> 容量/配置/权益
```

---

## 第七步：抽象成可复用视觉方法

每次学习完必须输出“不是这个图专属”的方法。

```text
可复用主体方法：
可复用特效字方法：
可复用卖点卡方法：
可复用背景方法：
可复用色彩方法：
可复用信息动线：
不建议复用的地方：
适合迁移的产品：
```

---

## 第八步：转成 AI 提示词方向

视觉学习最终要能转成可执行提示词。

```text
产品主体：
主体光感：
核心特效字：
背景设计：
卖点卡：
点缀色：
局部证据：
信息层级：
负面约束：
```

示例骨架：

```text
产品主体作为画面最亮、最清晰的核心，真实商业摄影质感，边缘高光，材质反光清楚。
核心卖点大字位于产品后方，使用与产品高光一致的同色系烟雾/光效字，不遮挡产品面板和Logo。
左侧深色卖点卡，白色高对比标题，少量橙金色编号点缀，清晰展示3个功能证据。
深色背景，有低亮度空间层次、金属轮廓、暖色底光和轻微星点，承托产品，不抢主体。
避免文字遮挡产品、背景过亮、特效字过实、色彩杂乱、卖点卡抢主体。
```

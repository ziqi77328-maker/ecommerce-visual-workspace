# 主图证据链与视觉层级检查表

用于检查一张优秀主图为什么能让用户快速相信卖点，并判断它的结构是否值得复用。

---

## 1. 主卖点是否有证据

主卖点不能只写一句口号，必须在画面里有可见证据。

```text
主卖点：
对应证据：
证据位置：
证据形式：局部放大 / 参数数字 / 材质展示 / 功能过程 / 对比结果 / 权益背书
用户能否一眼看懂：
```

示例：

```text
主卖点：0涂层精钢厚釜
对应证据：锅盖局部、内胆局部、材质标签、0涂层图标
证据形式：局部放大 + 参数标签 + 材质特写
```

---

## 2. 信息层级是否清楚

用 5 秒判断用户看到信息的顺序。

```text
第 1 秒：品牌/大卖点/产品主体
第 2 秒：核心参数或功能证据
第 3 秒：局部放大与使用过程
第 4 秒：容量/适用人群/赠品/权益
第 5 秒：是否形成点击理由
```

如果用户需要从小字里寻找主卖点，说明层级不够清楚。

---

## 3. 局部放大图是否服务主卖点

局部放大图不是装饰，必须承担证明作用。

| 局部内容 | 证明什么 | 是否必要 |
| --- | --- | --- |
| 锅盖/内胆结构 | 材质、厚度、无涂层 | 高 |
| 烹饪过程透视 | 火力、锁鲜、保温 | 高 |
| 控制面板 | 功能丰富、操作可信 | 中 |
| 赠品小图 | 权益丰富 | 中 |

---

## 4. 视觉冲击是否来自卖点本身

强视觉最好围绕核心卖点展开，而不是单纯加特效。

```text
大字是否是核心卖点：
特效是否强化主卖点：
产品主体是否清晰：
局部证据是否抢主体：
底部权益是否喧宾夺主：
```

---

## 5. 复用评分

每张案例学习完后，给它做复用评分。

| 项目 | 分数 | 判断标准 |
| --- | --- | --- |
| 卖点清晰度 | 1-5 | 第一眼能否看出卖什么 |
| 证据完整度 | 1-5 | 卖点是否有图像或参数支撑 |
| 视觉层级 | 1-5 | 信息顺序是否自然 |
| 产品清晰度 | 1-5 | 主体是否真实、完整、可辨认 |
| 可迁移性 | 1-5 | 是否能迁移到其它家电主图 |

建议只把总分 20 分以上的案例沉淀为长期方法。

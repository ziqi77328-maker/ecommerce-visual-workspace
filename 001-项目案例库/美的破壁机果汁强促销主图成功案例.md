## 本次有效方向

### 1. 先按 008 决策流程做生成前确认

这类主图不能直接生成，必须先走两道门槛：

```text
信息完整性确认
+
画面文字预览确认
```

本次有效点：

- 用户先提供产品图、主卖点、小卖点、价格和饮品氛围要求。
- 再补充容量和权益，并明确“信息均为确认的”。
- 生成前重新核对是否遗漏官方资料、型号、价格权益、去重、AI 感控制和出图后检查。
- 用户确认最终文字预览后再生成图片。

以后同类任务默认沿用这个流程。

### 2. 核心卖点只保留一个

本次同时出现“精钢6叶龙牙刀”和“一键高温洗”。

最终选择：

```text
主标题：精钢6叶龙牙刀
辅助卖点：一键高温洗
```

原因：

- 刀头更适合做破壁机第一眼差异化。
- 能和杯内水果、果汁旋涡、细腻口感形成视觉证据链。
- 高温洗是清洁顾虑解决点，适合做辅助卖点，不抢主标题。

### 3. 杯内果汁能让产品从“白底素材”变成“使用中主图”

破壁机如果只放干净产品图，画面会偏静态。

更有效的方式：

```text
透明杯内加入水果块和果汁液体
+
自然搅打旋涡
+
前景放 3-4 杯果汁 / 豆浆
```

注意：

- 果汁和水果只能强化使用结果，不能遮挡杯体结构、刀头区域、控制面板和 Logo。
- 周围饮品杯尺寸低于产品主体，只做氛围，不做主角。
- 饮品颜色可以用橙色、粉橙色、莓果色增强点击感。

### 4. 价格腰带按仓库红金强促销风格走

用户提供过粉色价格腰带参考，但最终确认按仓库风格。

有效结构：

```text
底部红金价格腰带
+
活动到手价
+
超大 ¥399
+
咨询客服更优惠
+
正品保障 送运费险
```

注意：

- 价格数字必须是价格区最大元素。
- 价格腰带高度控制在画面 14%-18%。
- 总成交区不超过 22%。
- 价格区只写成交权益，不写产品功能卖点。
- 权益只出现一次，不在底部重复。

### 5. 产品真实外观优先级最高

破壁机最容易出问题的是透明杯、把手、控制面板和 Logo 被改形。

提示词必须锁定：

```text
灰色机身
透明搅拌杯
右侧把手
顶部杯盖
正面数显控制面板
橙色屏显
按键布局
底部 Midea Logo
```

只允许优化：

- 商业摄影光影
- 轮廓高光
- 接触阴影
- 清晰度
- 杯内饮品氛围

不允许：

- 换型号
- 改颜色
- 改杯体结构
- 改把手
- 改控制面板
- 改 Logo

## 可优化点

- AI 生成中文和 Logo 仍需人工复核，正式上架建议用设计工具重排文字。
- 产品杯内果汁不能过满，否则会遮挡刀头和杯体结构。
- 证据卡不宜过多，本次只保留“6叶刀头结构示意”和“凹盘加热细节”即可。
- 如后续有官方型号、赠品、补贴或售后承诺，可以追加到价格权益区，但不能编造。

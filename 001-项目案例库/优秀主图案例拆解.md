# 优秀主图案例拆解

本文件基于一组美的厨房小家电京东主图案例，提炼可复用的高转化主图结构。

---

## 总体规律

这些案例不是单纯展示产品，而是在一张图里完成完整购买决策：

品牌信任 -> 核心卖点 -> 功能证据 -> 产品质感 -> 价格刺激 -> 权益保障

---

## 共性1：顶部品牌背书固定化

常见元素：

- Midea 品牌 Logo
- 京东新品
- 365天只换不修
- 平台活动标识
- 官方正品/售后保障

复用方法：

顶部左侧保留品牌背书区，高度控制在画面 8%-15%，不要压住产品主体。品牌区可以用蓝色、黑色或白色横条，和平台活动氛围保持一致。

---

## 共性2：左侧信息卡负责讲清购买理由

优秀案例常用左侧黑金/米白信息卡。

结构：

1. 顶部新品/旗舰/经典之选标签
2. 核心大卖点
3. 利益短句
4. 三行功能解释
5. 局部功能图或图标

可复用模板：

```text
【背书标签】
【核心大卖点】
【用户利益短句】

【功能1】+【小字解释】
【功能2】+【小字解释】
【功能3】+【小字解释】
```

示例：

```text
重磅新品
22分钟快煮饭
省时速享

24H安心预约 / 早期不匆忙，晚归不等待
抗菌白玉釜内胆 / 99.99%抗菌率，耐磨不粘
3L黄金容量 / 适合1-5人家庭使用
```

---

## 共性3：产品主体必须真实、清晰、占比大

案例里的产品通常占画面 45%-60%。

关键要求：

- 产品位于右侧或画面中心
- 正面略俯视，能看到顶部和控制面板
- 产品 Logo、按键、显示屏清晰
- 边缘有高光描边
- 材质表现明确：白色塑料、金属不锈钢、黑色镜面、晶釜纹理

复用方法：

提示词中必须明确：

```text
产品主体真实清晰，不变形，控制面板和品牌标识清晰可读，产品占画面50%-60%，边缘高光描边，真实商业摄影质感。
```

---

## 共性4：产品周围用徽章做快速判断

常见徽章：

- 2L / 3L / 4L 容量徽章
- 适合 1-5 人、2-6 人、3-8 人
- 316L 母婴级不锈钢
- 0 涂层
- 1 级能效 / 2 级能效
- 55℃ 一键保温
- 99.99% 抗菌率
- 立省金额

复用方法：

把徽章放在产品右上、右侧或局部放大旁边。徽章只表达一个快速判断，不承载长文案。

---

## 共性5：底部价格区是成交引擎

优秀案例底部通常不是普通价格，而是完整成交模块。

结构：

```text
活动标签 + 到手价大数字 + 返现/赠品 + 限时口号 + 售后/物流/价保
```

常见写法：

- 活动到手价 ¥169
- 晒单返现10元
- 赠配件全套
- 真正低价 限时促销
- 价保30天 买贵补差
- 京东速发 / 全国联保 / 官方正品

视觉规律：

- 价格数字最大
- 红金配色
- 火焰、流光、描边制造冲击
- 赠品用实物小图横向排列
- 权益用小图标快速识别

---

## 共性6：场景承担情绪价值

不同案例对应不同场景。

| 场景 | 适用卖点 | 视觉元素 |
| --- | --- | --- |
| 暖厨房 | 快煮饭、预约、家庭使用 | 木桌、米饭、家常菜、暖阳 |
| 简洁米白 | 新品、白玉晶釜、高端质感 | 花朵、米白背景、柔光 |
| 金属科技 | 316L、0涂层、能效 | 黑金背景、金属反光、流光 |
| 火焰促销 | 大促、降价、强火力 | 红金火焰、价格爆发、冲击字 |
| 花瓣晶体 | AI雷达、控温释香、养生 | 花瓣、晶体、金色光线 |

---

## 可复用版式1：左卡右品底价型

适合：

- 京东家电主图
- 信息密度较高的爆款图
- 需要同时讲卖点和价格

结构：

```text
顶部：品牌背书栏
左侧：黑金核心卖点卡 + 三行功能解释
右侧：产品主体 + 容量/材质徽章
底部：价格爆发栏 + 返现 + 赠品 + 售后权益
```

提示词骨架：

```text
京东家电主图，1:1正方形构图，高清4K商业摄影质感。顶部左侧品牌背书栏，品牌Logo清晰，带京东新品或365天只换不修徽章。左侧黑金核心卖点卡，金色描边，大标题突出【核心卖点】，下方写【利益短句】。左侧下方三行功能解释，每行包含圆形图标、主卖点和小字解释。右侧放置【产品】真实主视觉，正面略俯视，产品占画面50%-60%，控制面板和品牌标识清晰，边缘高光描边。容量角标靠近产品主体，位于产品视觉重心略偏上位置，只出现一次，右下角展示【局部证据】。底部红金价格爆发栏，超大数字【到手价】，搭配【返现】、【赠品】、【促销口号】和【售后权益】。整体信息密度高，层级清晰，京东爆款主图，高点击率，高转化。
```

## 可复用版式2：巨卖点产品冲击型

适合：

- 功能强卖点
- 黑金/火焰/科技促销风格
- 沥糖釜、316L、0涂层、强火力等强功能产品

结构：

```text
顶部：品牌背书
左上：超大核心卖点
中右：产品主体
右侧：局部内胆/材质证据
底部：VS价格或大促价格
```

关键词：

- 专业沥糖釜
- 控糖不降香
- 0涂层母婴级
- 316L不锈钢鼎釜
- 低糖健康饭
- 原价 VS 到手价

---

## 可复用版式3：花瓣晶体高端型

适合：

- AI雷达煲
- 实时控温
- 养生壶赠品
- 高端新品

结构：

```text
顶部：京东新品 + 品牌
左侧：竖向功能证明卡
右侧：产品主体
背景：花瓣、晶体、金色柔光
底部：预售价 + 晒单红包 + PLUS权益
```

关键词：

- 金色花瓣
- 透明晶体
- 柔光高端感
- AI实时控温
- 米汤分离
- 买就送养生壶

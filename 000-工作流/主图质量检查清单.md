# 主图质量检查清单

出图后按下面清单检查。

---

## 产品检查

- [ ] 产品没有变形
- [ ] 产品颜色准确
- [ ] 产品没有被改成其它型号；光影、角度和透视可以优化，但品牌、颜色、结构和关键识别部件不能改变
- [ ] 产品占画面 45%-60%
- [ ] 品牌 Logo 没有错乱
- [ ] 控制面板、按钮、刻度清晰
- [ ] 材质质感真实
- [ ] 配件、内胆、局部放大清晰可辨
- [ ] 低糖电饭煲出现“低糖饭/沥糖饭/控糖饭/米汤分离”时，画面必须出现沥糖釜/沥糖篮
- [ ] 低糖饭证据区必须是“上沥糖釜/沥糖篮 + 下配套内胆”组合，不能只用单个普通内胆或饭碗替代
- [ ] 微波炉/微烤炸一体机出现“不锈钢内胆/0涂层内胆/易洁内胆”时，证据区不能用锅、盘、碗或无关配件替代内胆

---

## 信息检查

- [ ] 第一眼能看到核心大卖点
- [ ] 核心卖点不超过 2 行
- [ ] 辅助卖点控制在 3-5 个
- [ ] 每个辅助卖点都有小字解释
- [ ] 容量、人群、功能徽章明确
- [ ] 价格权益模块已经出现；转化型主图不能只讲卖点和产品
- [ ] 价格数字足够大
- [ ] 返现、赠品、补贴、售后权益完整
- [ ] 未提供的参数、材质编号、检测数据、价格、返现、赠品和售后承诺没有被自动补写
- [ ] 微波炉/微烤炸一体机未明确提供 304 或 316L 时，没有自动写 304/316L

---

## 版式检查

- [ ] 顶部品牌背书不遮挡主体
- [ ] 左侧信息卡层级清晰
- [ ] 产品主体是画面视觉中心
- [ ] 价格区是独立视觉锚点
- [ ] 价格数字是价格区最大元素，价格区不是普通小字或小贴片
- [ ] 底部权益区优先写返现、赠品、补贴、售后、价保、配送等成交权益，没有重复左侧小卖点
- [ ] 如果使用左侧整合价格区，卖点、价格、权益在同一信息面板里形成连续阅读路径
- [ ] 火焰不是必选项；如使用火焰、流光或爆闪，没有遮挡产品主体和产品证据
- [ ] 文字、产品、贴片没有互相遮挡
- [ ] 背景没有抢主体
- [ ] 重要信息在缩略图里仍可识别

---

## 平台检查

- [ ] 适合 1:1 主图比例
- [ ] 适合手机端缩略图浏览
- [ ] 促销氛围符合平台活动
- [ ] 没有使用不确定或无法证明的绝对化表述
- [ ] 没有出现乱码、错字、错误品牌

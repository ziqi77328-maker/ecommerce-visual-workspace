# 主图质量检查清单

出图后按下面清单检查。

---

## 产品检查

- [ ] 产品没有变形
- [ ] 产品颜色准确
- [ ] 产品占画面 45%-60%
- [ ] 品牌 Logo 没有错乱
- [ ] 控制面板、按钮、刻度清晰
- [ ] 材质质感真实
- [ ] 配件、内胆、局部放大清晰可辨

---

## 信息检查

- [ ] 第一眼能看到核心大卖点
- [ ] 核心卖点不超过 2 行
- [ ] 辅助卖点控制在 3-5 个
- [ ] 每个辅助卖点都有小字解释
- [ ] 容量、人群、功能徽章明确
- [ ] 价格数字足够大
- [ ] 返现、赠品、补贴、售后权益完整

---

## 版式检查

- [ ] 顶部品牌背书不遮挡主体
- [ ] 左侧信息卡层级清晰
- [ ] 产品主体是画面视觉中心
- [ ] 价格区是独立视觉锚点
- [ ] 文字、产品、贴片没有互相遮挡
- [ ] 背景没有抢主体
- [ ] 重要信息在缩略图里仍可识别

---

## 平台检查

- [ ] 适合 1:1 主图比例
- [ ] 适合手机端缩略图浏览
- [ ] 促销氛围符合平台活动
- [ ] 没有使用不确定或无法证明的绝对化表述
- [ ] 没有出现乱码、错字、错误品牌

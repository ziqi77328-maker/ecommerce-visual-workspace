# 产品关键词匹配与优化规则

用于在用户给出一个产品后，精准匹配仓库中的场景、版式、卖点、价格权益和 AI 提示词，并进行转化优化。

---

## 一、先判断产品输入是否完整

每次生成关键词前，先确认以下信息：

- 品牌
- 类目
- 型号
- 产品颜色和外观
- 容量、功率、人数、能效等参数
- 已确认的核心卖点
- 已确认的辅助卖点
- 到手价、原价、返现、赠品、补贴、售后权益
- 平台和主图目标
- 是否有参考图或竞品图

如果用户没有明确提供，不要自动写入具体参数。例如：

- 未提供 304，不写 304 不锈钢
- 未提供 316L，不写 316L
- 未提供抗菌率，不写 99.9% 或 99.99%
- 未提供一级能效，不写一级能效
- 未提供低糖数据，不写还原糖降低百分比
- 提供低糖数据时，才把还原糖百分比做成数据可视化贴片

### 低糖电饭煲强制检查

如果产品定位、用户输入、参考图或主标题里出现“低糖饭、沥糖饭、控糖饭、米汤分离、低糖电饭煲、专业沥糖釜”等表达，必须先执行 [004-卖点表达库/C-沥糖饭.md](../004-卖点表达库/C-沥糖饭.md) 的强制规则。

生成关键词时必须写入：

```text
上方专业沥糖釜 / 沥糖篮，内盛米饭，圆孔沥水结构清晰
+
下方产品配套内胆，与沥糖釜形成上下组合
+
专业沥糖釜 / 米汤分离标签
```

禁止用下面内容替代低糖饭证据：

```text
单个316L不锈钢内胆
单个普通内胆盛米饭
单个饭碗
普通配件展示
```

316L 内胆只能作为材质卖点，不能作为低糖饭的唯一视觉证据。

## 二、类目到关键词库的匹配

### 电饭煲

优先匹配：

- 容量表达
- 快煮饭
- 白玉晶釜
- 沥糖饭
- AI 智能
- 预约功能
- 保温功能
- 0 涂层
- 316L 不锈钢

常用场景：

- 厨房场景
- 生活场景
- 简洁场景
- 特效场景中的白玉、花瓣、赤炎、科技、水流

推荐版式：

- 左文右图：适合多卖点、强价格、京东活动主图
- 居中主视觉：适合单一强卖点、新品、高端款

低糖电饭煲额外要求：

- 低糖饭、沥糖饭、控糖饭、米汤分离出现时，必须展示沥糖釜/沥糖篮。
- 证据区必须是“上沥糖釜/沥糖篮 + 下配套内胆”的组合。
- 沥糖釜圆孔、金属口沿、米饭颗粒和上下空间关系要清晰。
- 未提供还原糖数据时，不写降低比例。

### 电热水壶 / 养生壶

优先匹配：

- 容量表达
- 316L 不锈钢
- 大功率加热
- 预约功能
- 保温功能

常用场景：

- 简洁场景
- 厨房场景
- 生活场景中的养生生活、慢生活、品质生活
- 特效场景中的晶钛、珍珠、科技

### 微波炉

优先匹配：

- 容量表达
- 不锈钢内胆
- 易清洁
- 平板易洁
- 双旋钮好操作
- 火力调控
- 加热食物证据

常用场景：

- 厨房场景
- 简洁场景
- 强促销浅背景

优化重点：

- 门内加热食物和暖黄色工作光
- 产品结构、旋钮、把手、玻璃门不能变形
- 不臆造 304、316L 等材质编号
- 底部红金价格爆发栏增强成交感

### 空气炸锅 / 破壁机 / 电蒸锅 / 电磁炉

优先匹配：

- 容量表达
- 大功率加热
- 预约功能
- 保温功能
- 0 涂层
- AI 智能

常用场景：

- 厨房场景
- 生活场景中的美食展示、家庭用餐
- 特效场景中的赤炎、科技、粒子流光

## 三、卖点层级匹配

每张主图只设一个核心大卖点，辅助卖点控制在 3-5 个。

### 核心大卖点选择规则

优先选择最有差异化、最能直接影响购买的卖点：

1. 健康安全：0 涂层、316L、白玉晶釜、抗菌内胆
2. 效率速度：18 分钟快煮、22 分钟快煮、大功率速热
3. 功能升级：AI 雷达煲、智能控温、米汤分离
4. 容量适配：23L 超大容量、3L 黄金容量、适合多人家庭
5. 价格利益：活动到手价、立省金额、晒单返现、赠品

### 辅助卖点选择规则

辅助卖点负责补充购买理由，优先组合：

- 参数类：容量、功率、适用人数、能效
- 功能类：预约、保温、快煮、火力调节
- 材质类：不锈钢、0 涂层、白玉晶釜
- 证据类：内胆局部放大、控制面板放大、门内加热食物、配件展示
- 权益类：返现、赠品、价保、全国联保、京东速发

低糖饭证据优先级高于普通内胆证据。出现低糖饭时，先安排沥糖釜/沥糖篮与配套内胆的组合，再补充 316L、304 等材质信息。

### 卖点表达优化

弱表达要改成强表达：

- 大容量 -> 3L 黄金容量 / 23L 超大容量
- 加热快 -> 22 分钟快煮饭 / 大功率速热
- 好清洁 -> 易洁大平板 / 金属内胆易清洁
- 智能 -> AI 实时控温 / AI 识别一键煮
- 健康 -> 0 涂层健康内胆 / 米汤分离低糖饭

### 标题特效防同质化

生成主图前先选择标题视觉风格，不要默认所有主图都使用红金 3D 大字。

优先级：

- 金属质感大字：适合 0 涂层、不锈钢、精钢厚釜、金属内胆
- 火力热能大字：适合 360°旋风空气炸、大火力、速热、烘烤
- 科技数显大字：适合 数显大屏、智能菜单、一级能效
- 清透健康大字：适合 0 涂层、少油、健康、易清洁

备选：

- 数字主视觉：适合 23L、360°、365天，但不作为默认首发主标题
- 权益章：适合京仓直发、价保、只换不修，只作为辅助背书

## 四、场景匹配

### 简洁场景

适合：

- 产品质感强
- 参数不复杂
- 想突出产品真实外观
- 高端、低干扰、白底或浅背景主图

关键词：

- 简洁空间背景
- 大面积留白
- 低干扰环境
- 商业摄影布光
- 产品占画面 50% 以上
- 背景轻微虚化

### 厨房场景

适合：

- 厨房小家电
- 需要用户理解使用环境
- 电饭煲、微波炉、破壁机、空气炸锅等

关键词：

- 现代厨房场景
- 木质餐桌
- 厨房台面
- 热气米饭
- 家常菜摆盘
- 背景厨具虚化
- 暖色自然光

### 生活场景

适合：

- 养生、家庭、早餐、品质生活主题
- 需要情绪价值和用户代入感

关键词按目标选择：

- 温馨早餐
- 家庭用餐
- 养生生活
- 慢生活
- 品质生活
- 美食展示
- 阳光厨房

### 特效场景

适合：

- 强卖点放大
- 新品主推
- 大促主图
- 高点击率视觉

匹配规则：

- 白玉、健康、高端白色产品 -> 白玉空间、玉石纹理、晶体碎片
- 316L、0 涂层、金属内胆 -> 晶钛空间、金属高光、拉丝金属
- 快煮、大功率、IH、速热 -> 火焰背景、热浪效果、能量火焰
- AI、智能控温 -> 雷达扫描、数据流、蓝色科技光
- 沥糖饭、米汤分离 -> 水流分离、水波纹、透明水纹
- 高端柔和、女性向、养生 -> 花瓣环绕、透明晶体、柔光高端感

## 五、版式匹配

### 左文右图

优先用于：

- 卖点较多
- 价格权益需要完整表达
- 京东活动主图
- 产品右侧展示更清晰

结构：

- 顶部品牌背书栏
- 左侧黑金核心卖点卡
- 三行小卖点解释模块
- 右侧产品主视觉
- 右下角局部放大展示
- 底部红金价格爆发栏
- 赠品与返现权益模块

价格区也要做风格选择：

- 强促销：红金促销腰带
- 清透产品：白红平台补贴区
- 高端金属：黑金价格铭牌
- 领券返现：优惠券价格卡
- 秒杀直降：闪电直降条

### 居中主视觉

优先用于：

- 产品外观强
- 单一强卖点
- 新品首发
- 高端款或爆款冲击图

结构：

- 顶部品牌背书
- 上方超大核心卖点
- 中心产品主体
- 产品周围容量、材质、功能徽章
- 底部价格权益

## 六、价格区匹配

### 左下角价格区

适合左文右图，价格跟卖点形成整体。

关键词：

- 活动到手价
- 超大价格数字
- 底部红金促销栏
- 火焰价格
- 数字突破边框
- 限时促销

### 左侧整合价格区

适合卖点和价格都放左侧，右侧完整展示产品。

关键词：

- 左侧整合成交区
- 卡片式价格模块
- 卖点价格统一阅读节奏
- 红色价格主体
- 金色高光辅助

### VS 对比价格区

适合大幅降价、立省金额、原价对比。

关键词：

- 原价 VS 到手价
- 立省金额
- 价格差异放大
- 高冲击价格对比

### 右下角价格区

适合产品占比较大、左侧卖点较多、需要平衡画面重心。

关键词：

- 右下角价格锚点
- 红色价格模块
- 高对比价格数字
- 局部突破边框

## 八、优化检查规则

生成关键词后必须检查：

- 核心卖点是否第一眼可读
- 辅助卖点是否为 3-5 个
- 产品占比是否 45%-60%
- 产品结构、颜色、Logo、控制面板是否被明确约束
- 价格数字是否足够突出
- 赠品、返现、售后是否靠近价格区
- 背景是否服务产品，而不是抢主体
- 是否出现用户没有提供的参数
- 是否加入负面关键词防止变形、乱码、错品牌
- 如果出现低糖饭、沥糖饭、控糖饭或米汤分离，是否已经强制加入沥糖釜/沥糖篮 + 配套内胆组合
- 是否避免用单个普通内胆或单个 316L 内胆替代低糖饭证据

```text
核心卖点：
辅助卖点：
产品证据：
价格权益：
场景关键词：
特效关键词：

3. 优化后的主图文案
主标题：
副标题：
三行卖点：
价格区：
权益区：

4. 完整 AI 提示词
可直接复制使用的中文提示词。

5. 负面关键词
防止产品变形、文字乱码、参数臆造、品牌错误。

6. 优化说明
说明为什么这样匹配，以及哪些信息因用户未提供而没有写入。
```

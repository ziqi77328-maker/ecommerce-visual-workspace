# 000-工作流

这里把各个素材库串成可复用流程。

建议每次做主图都按这个顺序走：

1. 填写产品信息
2. 选择主图类型
3. 选择版式结构
4. 选择场景氛围
5. 组合核心卖点
6. 组合价格权益
7. 生成 AI 提示词
8. 检查输出质量
9. 复盘案例并沉淀新模板

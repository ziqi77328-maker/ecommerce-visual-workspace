# 000-工作流

这里把各个素材库串成可复用流程。

建议每次做主图都按这个顺序走：

1. 先用美云销获取官方商品资料、主图和详情页素材
2. 用 `008-agent决策中心` 判断主图目标、版式、卖点优先级、固定素材和 AI 感风险
3. 填写产品信息
4. 选择主图类型
5. 选择版式结构
6. 选择场景氛围
7. 组合核心卖点
8. 组合价格权益
9. 生成 AI 提示词
10. 检查输出质量
11. 把失败原因写入失败数据库
12. 复盘案例并沉淀新模板

# 000-工作流

这里把各个素材库串成可复用流程。

建议每次做主图都按这个顺序走：

1. 先用美云销获取官方商品资料、主图和详情页素材
2. 填写产品信息
3. 选择主图类型
4. 选择版式结构
5. 选择场景氛围
6. 组合核心卖点
7. 组合价格权益
8. 生成 AI 提示词
9. 检查输出质量
10. 复盘案例并沉淀新模板

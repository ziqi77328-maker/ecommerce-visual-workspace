# 常用图片素材命名规范

> 本规范先建立标准编号和调用规则，不改动现有图片文件名，不压缩、不替换、不移动 `常用图片` 文件夹里的图片。

## 命名原则

- 图片原文件名保持不变，避免破坏历史提示词、案例复盘和固定素材引用。
- 新增素材时优先使用“中文可读名 + 序号”的方式，方便人工识别。
- 每张素材在 README 中登记一个稳定素材编号，后续可以用编号检索，不依赖改文件名。
- 素材编号只用于文档、提示词和检索，不代表真实文件名。

## 素材编号格式

```text
asset-[类目]-[卖点]-[类型]-[序号]
```

示例：

```text
asset-ricecooker-0coating-innerpot-01
asset-ricecooker-316-sticker-01
asset-ricecooker-low-sugar-drainer-01
```

## 新增素材登记要求

新增图片进入 `常用图片` 时，同步补充：

- 素材编号
- 当前文件名
- 对应卖点
- 使用场景
- 是否必须使用原图
- 是否允许裁切、缩放、加投影
- 禁止事项

## 禁止事项

- 不压缩现有图片。
- 不替换现有图片。
- 不改色、不重绘、不重采样。
- 不移动现有图片到其它目录。
- 不在没有同步修改引用的情况下改文件名。
- 不用新编号替代真实文件路径；生成时仍应明确写出当前文件名。

# 常用图片素材调用规则

这个文件夹里的图片是主图生成时的固定证据素材。凡是命中对应卖点，优先使用这里的原图素材，不重新绘制同类文字贴片。

素材命名和编号规则见 [素材命名规范.md](素材命名规范.md)。当前图片文件名保持不变，不压缩、不替换、不移动。

## 标准组合关系

### 低糖电饭煲

大部分低糖电饭煲主图里，右下角优先使用：

```text
上方：沥糖釜.jpg，里面装香香软软的白米饭
+
下方：用户提供或常用图片中的标准电饭煲内胆
+
旁边：对应材质贴片
```

这个组合必须保持上釜下胆的空间关系。不要左右并排成两个孤立配件，不要只放饭碗或单个内胆。

### 晶钛和0涂层内胆

晶钛内胆、0涂层内胆在主图里优先作为产品旁边或右下角证据图出现：

- 内胆保持完整口沿、内壁、材质标识和真实比例。
- 底部加入金色光圈、环绕流光、橙色热力火焰、轻微蒸汽和真实投影。
- 材质贴片靠近内胆边缘，但不能遮挡米饭、内胆口沿和材质标识。
- 不只用文字表达晶钛或0涂层，必须有内胆图片证据。

## 素材保护要求

- 不拉伸、不压扁、不改变原始比例。
- 不改色、不套过强滤镜，不把银色材质改成金色、黑色或彩色。
- 不重新生成素材上的文字，不让文字变糊、乱码或缺字。
- 可以做等比例缩放、轻微投影、轻微外发光、边缘高光和透明背景融合。
- 素材不能被价格腰带、产品主体、火焰、光效或其它贴片遮挡关键信息。
- 如果素材原图带文字，最终图里文字必须清晰可读。

## 生成提示词固定补充

命中对应卖点时，在提示词中加入：

```text
使用仓库常用图片文件夹中的固定素材作为产品证据贴片，保持素材原始比例、原始颜色和文字清晰度，不重绘、不变形、不变色、不模糊。
```

低糖电饭煲必须额外加入：

```text
低糖饭证据必须使用常用图片/沥糖釜.jpg，保持原图形状、金属颜色、孔位结构和文字清晰，不用普通内胆或饭碗替代。
```

304/316L 材质必须额外加入：

```text
304 不锈钢或 316L 不锈钢卖点必须使用常用图片文件夹中的对应材质贴片素材表达，不生成单独文字贴片。
```

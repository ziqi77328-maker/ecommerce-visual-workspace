# 微波炉-京东主图-强促销浅背景学习方向

## 定位

本文件不是最终版提示词，而是一次微波炉主图生成复盘后的学习方向。

适用于后续继续优化京东家电微波炉主图时参考，尤其是黑色机身、双旋钮、平板易清洁、微烤二合一、光波烧烤、除菌、价格促销类主图。

核心学习点：

- 黑色产品必须突出，不能和背景融为一体
- 画面需要有京东大促电商冲击力，不能太素
- 主标题必须比小卖点更醒目，颜色、字号、特效都要拉开层级
- 微波炉门内要有“正在加热食物”的使用证据
- 微烤炸一体机可以用“开门不锈钢内胆证据图”强化 0 涂层不锈钢内胆
- 黑色产品需要清透高光、玻璃高光、金属扫光，避免灰暗发闷
- 底部价格区右侧应放权益信息，不要重复小卖点
- AI 主要生成画面结构、产品氛围、光效和占位，正式中文建议后期人工叠加

## 关键词方向

```text
京东家电微波炉主图，1:1正方形电商主图，高点击率，高转化，强电商促销视觉。

核心目标：
突出黑色美的微波炉产品主体，同时增强电商促销冲击力。背景不能使用纯黑、深黑或暗黑科技背景。使用浅灰、暖白、银灰或浅米色现代厨房/棚拍背景，让黑色产品形成强对比。画面加入红色、金色、蓝色促销光效和电商装饰元素，整体丰富但不杂乱。

产品主体：
保留黑色双旋钮微波炉外观，正面略俯视三分之二视角，右侧双旋钮、竖向把手、玻璃门、品牌Logo区域清晰可见。产品占画面55%-62%，比例真实，不变形。产品边缘有浅色轮廓光，顶部和侧面有柔和高光，底部有真实投影，黑色机身从浅色背景中清晰跳出。

关键使用证据：
微波炉玻璃门保持真实半透明，不要完全不透明，也不要变成完全展示窗。玻璃门内可隐约看见正在加热的食物，例如牛肉、蔬菜、米饭或家常饭菜。内部加入暖黄色加热光，光线从门内透出，表现微波炉正在工作。食物只作为“日常热饭热菜”的购买理由，不要抢产品主体。

主标题设计：
顶部中间大标题占据第一视觉层级。主标题内容为：微烤二合一 三餐风味烹饪。
“微烤二合一”必须最大最醒目，使用金色立体大字、红色描边、白色高光、轻微投影、金属质感。
“三餐风味烹饪”作为第二层级副标题，使用白色或浅黄色字体，带深色描边。
主标题必须比小卖点大2-3倍，不能和小卖点同色同粗细。标题后方可加入红金爆发光、斜切色块、速度线或促销光效，但不能遮挡产品。

小卖点区：
标题下方放三枚横向卖点徽章：
大平板易清洁 / 双管光波烧烤 / 微波高效除菌。
小卖点使用蓝白、白底红边或浅色徽章，与主标题颜色明显区分。字体比主标题小，整齐清晰，不抢主标题。

版式结构：
顶部左侧为品牌LOGO区。
顶部中间为强冲击主标题。
标题下方为三枚小卖点徽章。
画面中间偏右为微波炉产品主体。
产品右上方或左侧加入 C20 内胆参考小型局部放大窗。
产品右下或底部旁边展示烤架配件小图，并标注：下单配烤架。
底部为红金价格权益成交栏。

底部价格区：
底部使用京东红 + 金色描边价格栏，面积控制在画面底部18%-22%，不要压住产品。加入少量火焰、闪电、流光、爆发形状，增强抢购感。
突出文字：到手不高于 399。
其中 399 最大最醒目，使用红色或金色立体数字，白色描边，强投影。
辅助权益标签：
京仓速发 / 价保30天 / 品牌旗舰店 品质保障 / 京东多地 送货到家。

色彩搭配：
浅灰、暖白作为背景主底色。
红色用于促销和价格。
金色用于主标题重点字和品质感。
蓝色用于科技卖点徽章。
暖黄色用于微波炉内部加热光。
整体色彩丰富，有电商冲击力，但层级必须清楚。

背景与道具：
现代厨房台面或电商棚拍厨房环境。可加入少量餐盘、熟食、面包、饮料或厨房器具作为生活场景点缀，但必须弱化处理，不能抢产品主体。道具数量少，背景干净、有层次、有促销氛围。

重要限制：
中文文案不要完全依赖 AI 直接生成。AI 可生成版式、占位、氛围和大致文字效果，正式主图建议后期人工叠加准确中文。
```

## 微烤炸一体机补充方向

适用于黑色微烤炸一体机、微波炉空气炸一体机、带不锈钢内胆的微蒸烤炸产品。

```text
画面整体更清透明亮，使用浅银灰、暖白、浅色厨房棚拍背景。黑色机身要有玻璃门高光、镜面反射、白色轮廓光，银色把手要有金属扫光。

左下角或产品旁边放开门不锈钢内胆证据图：门向左打开，露出亮银色不锈钢内胆，后壁圆形风扇孔，两侧密集散热孔，内部玻璃转盘，右侧控制面板可见。这个模块用于证明“0涂层不锈钢内胆”，不要生成锅、盘或无关配件。

底部红金价格腰带右侧放权益条，而不是重复产品功能。推荐权益文案：“京仓直发 送货上门 / 价保618 早买早享受”。视觉可用红粉背景、白粉胶囊条、粉色爱心图标。
```

## 复盘要点

这一方向相比上一版优化重点：

- 背景从深黑改为浅灰/暖白，解决黑色产品不突出的问题
- 增加红金蓝促销色彩，避免画面太素
- 主标题使用金色立体字、红描边、白高光、投影，拉开和小卖点的层级
- 微波炉玻璃门内加入饭菜和暖黄色加热光，强化“日常热饭热菜”的购买理由
- 底部价格栏使用京东红金促销风格，突出 399 成交刺激
- 明确中文文案建议后期人工叠加，避免 AI 乱码影响正式主图

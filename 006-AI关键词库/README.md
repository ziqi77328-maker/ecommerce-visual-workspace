# 006-AI关键词库 — 通用AI关键词模板

> 用途：快速生成电商产品高质量图片、海报、主图、场景图
> 注意：可直接在生成工具（如 ComfyUI、Stable Diffusion、MidJourney 等）中调用

---

## 使用顺序

1. 先填写 [../000-工作流/主图信息填写表.md](../000-工作流/主图信息填写表.md)。
2. 从 [006-AI关键词库.md](006-AI关键词库.md) 选择基础画面、产品、场景、文案关键词。
3. 如果是京东活动主图，直接套用 [007-京东高转化主图模板.md](007-京东高转化主图模板.md)。
4. 根据产品实际卖点替换标题、参数、价格、赠品。
5. 加入负面关键词，减少产品变形、文字乱码、Logo错误。

---

## 提示词拼接顺序

```text
平台/画幅
+
产品主体
+
场景氛围
+
版式结构
+
品牌背书
+
核心卖点
+
辅助卖点
+
产品证据
+
价格权益
+
视觉风格
+
负面关键词
```

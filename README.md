# ecommerce-visual-workspace

电商主图视觉工作台，用于沉淀家电类目主图案例、卖点表达、价格权益、场景关键词和 AI 生成提示词。

当前重点场景：京东家电主图、平台大促与日常活动主图、电饭煲/电热水壶/厨房小家电高转化海报。

---

## 快速使用

1. 先看 [000-工作流/主图生成工作流.md](000-工作流/主图生成工作流.md)，确定要做哪一种主图。
2. 用 [000-工作流/主图信息填写表.md](000-工作流/主图信息填写表.md) 填产品信息、卖点、价格、权益。
3. 用 [000-工作流/产品关键词匹配与优化规则.md](000-工作流/产品关键词匹配与优化规则.md) 精准匹配场景、版式、卖点、价格区和 AI 关键词。
4. 从 [003-产品结构排版库](003-产品结构排版库) 选择版式。
5. 从 [002-产品场景库](002-产品场景库) 选择背景氛围。
6. 从 [004-卖点表达库](004-卖点表达库) 组合核心卖点和辅助卖点。
7. 从 [005-价格与权益表达库](005-价格与权益表达库) 选择价格区。
8. 用 [006-AI关键词库](006-AI关键词库) 生成完整提示词。
9. 参考 [001-项目案例库/优秀主图案例拆解.md](001-项目案例库/优秀主图案例拆解.md) 做复盘和优化。
10. 学习外部优秀案例或竞品图时，使用 [007-案例学习与卖点提取系统](007-案例学习与卖点提取系统) 提取卖点、时间表达和可复用公式。

新增经验沉淀时，先看 [000-工作流/仓库内容归类与更新规则.md](000-工作流/仓库内容归类与更新规则.md)，避免把同一段内容重复放到多个库里。

## 内容分工

- 工作流：记录怎么用仓库、怎么检查、怎么更新。
- 案例库：记录一次主图从初版到修改的过程和复盘。
- 场景库：记录背景、光效、氛围和材质空间。
- 结构库：记录左文右图、居中主视觉、价格区位置等版式。
- 卖点库：记录卖点文案、图形表达和用户利益解释。
- 价格权益库：记录到手价、返现、赠品、价保、只换不修等成交模块。
- AI关键词库：只放可以直接复制使用的组合提示词和负面关键词。
- 案例学习与卖点提取系统：记录如何从外部优秀案例中提取结构、卖点、时间表达和可复用公式。

## 高转化主图通用公式

品牌背书区
+
核心大卖点
+
3-5 个辅助购买理由
+
真实清晰产品主视觉
+
容量/材质/功能证据贴片
+
价格爆发区
+
返现/赠品/保障/售后权益

## 优秀案例共性

- 顶部固定品牌背书：品牌 Logo、京东新品、365 天只换不修、平台活动。
- 左侧承载购买理由：黑金卖点卡、功能图标、局部功能截图、参数解释。
- 右侧承载产品信任：产品占画面 45%-60%，面板、Logo、材质细节清晰。
- 底部承载成交动作：超大价格、限时活动、晒单返现、赠品、PLUS/补贴/售后。
- 用徽章补充快速判断：容量、人群、能效、材质、抗菌率、保温温度。
- 用场景建立价值感：暖厨房、餐桌米饭、花瓣/晶体/火焰/金属高光。

## 学习优秀主图视觉表达

学习优秀案例时，除了拆卖点，还要记录视觉表达过程：

```text
产品主体为什么亮眼
特效字为什么不抢主体
左侧卖点卡为什么清晰
背景为什么有设计但不杂乱
主色、辅色、点缀色如何分工
信息动线如何从吸引走向信任和购买理由
```

具体流程见 [007-案例学习与卖点提取系统/07-视觉表达学习流程.md](007-案例学习与卖点提取系统/07-视觉表达学习流程.md)。

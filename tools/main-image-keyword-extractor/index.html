背景配件勾选
背景 / 产品质感学习要点
  • 深色背景不是纯黑，要有深灰、黑金、石材纹理、暗金渐变或斜向光纹。
  • 产品后面要有柔亮背光圈，只亮产品周围，把轮廓托出来。
  • 产品质感靠反光，不靠磨皮：金属拉丝、玻璃反射、面板高光、边缘白色轮廓光要清楚。
  • 卖点框小而精，黑底金边、分层小条、证据小图，不做大表格。
  • 装饰物只服务产品：勾了什么就只生成什么，未勾选的果汁、豆浆、水果等不要出现。

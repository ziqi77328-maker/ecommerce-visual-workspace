# 电商主图关键词提取器

纯静态网页工具，不依赖外部接口或 CDN，可直接在中国境内浏览器中打开使用。

入口：

```text
index.html
```

能力：

- 按 `008-agent决策中心` 做信息完整性检查。
- 自动输出已提供、缺少信息、不能编造项和固定素材命中。
- 产品图、配件图、参考图、证据图分开上传，各支持最多 6 张本地预览。
- 卖点展示默认 1 条，可按需增加，并支持本地规则版“AI一键优化”。
- 支持选择左右构图或上下构图。
- 支持主标题、副标题、背景策略、背景配件勾选和自定义输入。
- 支持小卖点图标/卖点图片、卖点边框主色调、卖点背景板模式和额外附加提示词。
- 支持 VS 价格对比腰带。
- 支持电商预警，提醒标题过长、卖点过多、证据区过满、背景太复杂等问题。
- 生成画面文字预览。
- 生成关键词提取结果。
- 生成可复制 AI 出图提示词。
- 生成空白版后期排版提示词：去掉广告文案、顶部品牌背书和底部价格腰带，只保留产品、卖点边框留白和右侧参数小贴片文字。
- 内置“美的破壁机水果果汁强促销主图”成功案例。

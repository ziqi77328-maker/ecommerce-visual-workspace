# 中国境内网页版部署说明

这个工具是纯静态网页，不依赖外部接口、CDN、npm 或国外服务。部署后换电脑也可以通过网址使用。

## 推荐方式

优先使用国内对象存储静态网站：

- 阿里云 OSS 静态网站托管
- 腾讯云 COS 静态网站
- 华为云 OBS 静态网站

如果需要访问更快，可以再开启国内 CDN，并绑定已备案域名。

## 要上传的文件

上传这个目录里的 3 个文件到对象存储根目录：

```text
index.html
styles.css
app.js
```

也可以直接上传发布包：

```text
main-image-keyword-extractor-cn.zip
```

## 阿里云 OSS 简要流程

1. 创建 Bucket，地域选中国大陆。
2. 上传 `index.html`、`styles.css`、`app.js` 到 Bucket 根目录。
3. 开启静态网站托管。
4. 默认首页设置为 `index.html`。
5. Bucket 读权限按需要设置为公共读，或通过 CDN 访问。
6. 复制静态网站访问地址，即可在其它电脑打开使用。

## 腾讯云 COS 简要流程

1. 创建存储桶，地域选中国大陆。
2. 上传 `index.html`、`styles.css`、`app.js` 到存储桶根目录。
3. 开启静态网站功能。
4. 索引文档设置为 `index.html`。
5. 访问权限按需要设置为公有读，或通过 CDN 访问。
6. 复制静态网站访问地址，即可在其它电脑打开使用。

## 注意事项

- 图片上传只在浏览器本地预览，不会传到服务器。
- 草稿保存在当前浏览器的 localStorage，换电脑不会同步草稿。
- 如果要用自己的域名，在中国大陆稳定访问通常需要域名备案。
- 如果只给公司内部使用，也可以放在公司内网服务器、NAS 或宝塔面板静态站点里。

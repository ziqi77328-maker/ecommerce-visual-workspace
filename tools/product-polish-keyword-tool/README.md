# 产品图精修关键词匹配器

纯静态网页工具，不依赖外部接口或 CDN，可直接打开使用。

入口：

```text
index.html
```

定位：

- 只输出产品美化、背景场景、真实约束和负面关键词。
- 不输出价格、标题、卖点卡、促销腰带、优惠券、广告文字、品牌背书栏等海报排版内容。
- 适合把白底图、透明底产品图、普通产品图优化成更干净、更真实、更有场景关系的产品主视觉。
- 默认强制加入“产品质感增强”模块，要求保留材质微细节、边缘厚度、接缝、倒角、暗部层次、金属拉丝、玻璃厚度、真实接触阴影和商业摄影精修质感。

已接入的仓库图片规则：

- `002-产品场景库`：简洁、厨房、生活、科技、黑金、深色高端棚拍等场景氛围。
- `006-AI关键词库/006-AI关键词库.md`：基础画面参数、产品展示细节、材质、光影和负面关键词。
- `008-agent决策中心/04-AI感降低策略.md`：真实商品图约束、商业摄影棚布光、真实接触阴影、AI 感降低和负面约束。

未接入：

- 产品结构排版库
- 价格权益表达库
- 卖点卡、标题、促销腰带、品牌背书栏等海报排版内容

核心输入：

- 产品类目
- 上传参考图
- 优化目标
- 产品材质
- 背景风格
- 产品与背景关系
- 产品角度
- 产品状态，多选；包含开盖展示、不加锅盖、加入米饭、加入食物、火锅食材分区、杯内果汁/水果等状态约束
- 是否需要配件
- 产品位置、占比、留空

输出：

- 产品美化关键词
- 背景场景关键词
- 真实约束关键词
- 负面关键词
- 完整可复制提示词

# 003-产品结构排版库

用于沉淀主图版式结构、信息区域分布、产品占比、证据贴片位置和成交区布局。

## 使用顺序

1. 先看 [通用主图信息层级.md](通用主图信息层级.md)，确认品牌、卖点、产品、证据和价格区的基础层级。
2. 再按主图目标选择具体版式。
3. 最后回到 [../008-agent决策中心](../008-agent决策中心) 检查卖点是否重复、产品是否被遮挡、价格区是否过重。

## 分工

- 通用层级：所有主图共同遵守的品牌、卖点、产品、证据、价格区规则。
- 具体版式：只记录该版式的差异、比例、适用场景和可复用提示词。

# 008-agent决策中心

用于主图生成前的判断、锁定、风险拦截和失败复盘。

## 使用顺序

1. [01-主图决策树.md](01-主图决策树.md)：判断资料是否完整、主图目标、版式、核心卖点、证据区和生成门槛。
2. [02-优先级评分系统.md](02-优先级评分系统.md)：给卖点、版式、素材和风险做取舍，避免什么都放。
3. [05-强制锁定清单.md](05-强制锁定清单.md)：每次生成或改版前强制锁定产品外观、容量角标、卖点分区、证据图、配件和版式/价格区。
4. [04-AI感降低策略.md](04-AI感降低策略.md)：降低塑料感、梦幻感、文字乱码和证据素材变糊。
5. [03-失败数据库.md](03-失败数据库.md)：记录失败类型、原因和下次拦截规则。

## 硬性规则

- 没有完成强制锁定清单，不允许进入画面文字预览。
- 没有完成画面文字预览确认，不允许生成图片。
- 每次换排版、换背景、增删证据图或重新出图，都要重新执行强制锁定清单。

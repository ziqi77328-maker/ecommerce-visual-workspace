# 优先级评分系统

用于让 agent 在多个卖点、版式、素材和风险之间做取舍。

评分不是为了复杂化流程，而是为了避免“什么都想放，最后什么都不突出”。

---

## 总分结构

每次生成前按 100 分评估。

| 模块 | 分值 | 判断重点 |
| --- | --- | --- |
| 产品真实性 | 25 | 外观、结构、Logo、固定素材是否真实 |
| 卖点转化力 | 20 | 核心卖点是否强、是否能被证明 |
| 价格权益 | 15 | 到手价、返现、赠品、补贴、售后是否形成成交闭环 |
| 信息层级 | 15 | 标题、辅助卖点、证据、价格是否分区清楚 |
| 缩略图点击力 | 10 | 手机端第一眼是否能看懂 |
| 真实商业摄影感 | 10 | 是否减少 AI 感、塑料感和过度特效 |
| 可复盘性 | 5 | 是否记录成功点和失败点 |

低于 80 分不建议进入最终图。

---

## 卖点评分

每个候选卖点按 5 个维度打分，每项 0-5 分。

| 维度 | 高分标准 |
| --- | --- |
| 差异化 | 竞品少见，或在该价位更有优势 |
| 购买推动 | 能直接影响下单、降低顾虑 |
| 证据可视化 | 能用内胆、配件、局部、参数、过程图证明 |
| 真实性 | 有官方资料或用户确认 |
| 缩略图可读性 | 手机端小图也能识别 |

总分最高者做核心卖点。第二到第五名可做辅助卖点。

容量、适用人数这类参数通常只做贴片，不进入核心卖点竞争。

---

## 版式评分

| 版式 | 适配条件 | 加分 |
| --- | --- | --- |
| 左文右图 | 多卖点、强价格、京东活动图 | +5 |
| 居中主视觉 | 单强卖点、高端质感、新品 | +5 |
| 强促销浅背景左文右品 | 大促、补贴、低价、返现 | +5 |

扣分项：

- 产品主体低于画面 45%：-5
- 价格区不明显：-5
- 证据区遮挡产品：-5
- 左侧卖点和右侧贴片重复：-5
- 背景强过主体：-5

---

## 素材优先级

素材使用顺序：

1. 官方商品图、官方详情页素材
2. `常用图片` 中已有固定证据素材
3. 用户提供参考图
4. 案例库中已验证的构图方式
5. AI 生成的背景和装饰元素

关键证据不应由 AI 重新发明。

必须优先调用的固定素材：

| 卖点 | 优先素材 |
| --- | --- |
| 低糖饭/沥糖饭 | `常用图片/沥糖釜.jpg` |
| 304 不锈钢 | `常用图片/304不锈钢材质贴片1.png` 或 `常用图片/304不锈钢材质贴片2.jpg` |
| 316L/316 母婴级 | `常用图片/316母婴级不锈钢材质贴片1.png` 或 `常用图片/316母婴级不锈钢材质贴片2.png` |
| 0 涂层内胆 | `常用图片/0涂层电饭煲内胆1.png` 或 `常用图片/0涂层电饭煲内胆 特效.png` |
| 晶钛内胆 | `常用图片/晶钛电饭煲内胆1.png` |

## 风险评分

每个风险 0-5 分，分数越高越危险。

| 风险 | 触发条件 |
| --- | --- |
| 产品变形 | 产品结构、颜色、面板、Logo 不稳定 |
| 参数臆造 | 没有资料却写具体数字、等级或权益 |
| 证据错误 | 用错误配件证明卖点 |
| AI 感过重 | 背景、材质、字体、光影明显不真实 |
| 信息拥挤 | 主标题、贴片、价格、证据互相抢 |
| 缩略图不可读 | 手机端看不清核心卖点和价格 |

风险总分超过 10 分，必须先修正提示词或版式。

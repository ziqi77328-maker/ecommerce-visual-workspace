# 主图决策树

用于让 agent 在生成前做选择，而不是直接拼接所有关键词。

---

## 第一步：判断资料是否足够

每次用户给出产品资料、卖点、活动信息或参考图后，agent 必须先对照本步骤检查缺少哪些内容。

### 强制确认门槛 1：内容完整性确认

如果用户只提供了一部分卖点、价格、画面要求或产品图，agent 必须先向用户确认：

```text
你给的是全部要上图的信息，还是只是一部分？
是否还需要补充：型号、容量、主卖点、辅助卖点、价格权益、赠品、补贴、售后、活动标签、人群、场景？
```

用户没有明确确认前，禁止生成图片。

如果缺少下面任一项，先向用户核对或使用明确占位符，不直接进入最终生成：

- 品牌、类目、型号
- 产品真实外观图或官方主图
- 核心卖点
- 价格或价格占位符
- 至少 2 个成交权益或权益占位符
- 需要调用的固定素材

如果涉及具体参数，必须确认来源：

```text
容量 / 功率 / 材质编号 / 抗菌率 / 降糖率 / 能效等级 / 到手价 / 返现金额 / 赠品 / 售后承诺
```

未确认时只写结构，不写具体数字。

### 缺失信息核对输出

生成前必须输出，并等待用户确认：

```text
已提供信息：
缺少信息：
需要向用户核对的问题：
可以先用占位符处理的信息：
不能编造的信息：
是否确认当前信息已经足够进入文字预览：等待用户确认
```

缺少关键信息时，要先和用户核对。尤其是下面内容，不能自动补写：

```text
材质编号 / 检测数据 / 降糖比例 / 抗菌率 / 到手价 / 返现 / 赠品 / 补贴 / 售后承诺 / 型号结构 / 产品颜色
```

---

## 第二步：判断主图目标

| 输入特征 | 主图目标 | 首要任务 |
| --- | --- | --- |
| 有低价、返现、赠品、补贴 | 活动促销 | 价格权益必须突出 |
| 新品、首发、升级款 | 新品首发 | 品牌背书和核心卖点突出 |
| 卖点需要解释原理 | 功能教育 | 证据区优先 |
| 高客单、材质卖点强 | 高端质感 | 产品质感和真实光影优先 |
| 需要生活方式感 | 场景种草 | 使用场景和情绪价值优先 |

如果同时命中多个目标，优先级如下：

```text
活动促销 > 功能教育 > 新品首发 > 高端质感 > 场景种草
```

电商主图最终都要保留成交闭环。即使目标是功能教育，也不能缺少价格权益区。

---

## 第三步：选择版式

### 左文右图

优先用于：

- 多卖点
- 京东活动主图
- 需要价格爆发栏
- 产品主体适合放右侧

输出结构：

```text
顶部品牌背书
+ 左侧核心标题和辅助卖点
+ 右侧真实产品主体
+ 右侧或下方产品证据
+ 底部价格权益
```

### 居中主视觉

优先用于：

- 单一强卖点
- 高端质感
- 新品首发
- 产品外观本身有吸引力

输出结构：

```text
顶部品牌背书
+ 中心产品大主体
+ 周围少量功能贴片
+ 底部价格权益
```

### 强促销浅背景左文右品

优先用于：

- 大促活动
- 价格和权益很强
- 需要手机端缩略图强点击
- 产品颜色偏浅，需要干净背景托出

输出结构：

```text
浅色商业摄影背景
+ 左侧强卖点与价格利益
+ 右侧清晰产品和证据贴片
+ 底部红金价格腰带
```

---

## 第四步：选择核心卖点

只选 1 个核心卖点。

优先级：

1. 有强差异化且能被图片证明的卖点
2. 能直接降低购买顾虑的卖点
3. 能提升点击欲望的效率或健康卖点
4. 有价格利益但产品卖点不强时，价格作为主驱动
5. 容量只做产品主体右侧上半部信息区的小型角标，不作为核心标题

常见判断：

| 产品信息 | 核心卖点选择 |
| --- | --- |
| 低糖饭、沥糖饭、控糖饭 | 专业沥糖釜/米汤分离 |
| 316L、304、0 涂层、晶钛 | 材质健康安全 |
| 快煮、速热、大功率 | 效率速度 |
| AI、智能菜单、控温 | 智能功能 |
| 大额补贴、返现、赠品 | 活动利益 |

---

## 第五步：安排证据区

证据区优先级高于装饰。

必须遵守：

- 低糖饭必须出现沥糖釜/沥糖篮，不能用普通内胆或饭碗替代。
- 304/316L 材质优先调用 `常用图片` 固定贴片，不重新生成文字贴片。
- 0 涂层、晶钛、内胆卖点优先调用固定内胆素材。
- 微波炉/微烤炸一体机的不锈钢内胆证据必须表现真实内胆结构。

如果证据素材和背景特效冲突，删特效，保留证据。

---

## 第六步：判断是否容易出现 AI 感

命中以下情况时，必须执行 [04-AI感降低策略.md](04-AI感降低策略.md)：

- 背景过度梦幻、过度虚化、过多粒子流光
- 产品边缘像塑料、金属像假反光
- 字体像生成字、立体字过厚、红金特效过满
- 人手、食物、蒸汽、火焰不真实
- 产品和证据素材之间光源不一致
- 固定素材被重绘、文字变糊或比例变形

---

## 第七步：生成前最终判断

### 强制确认门槛 2：强制锁定清单确认

画面文字预览前，必须先执行 [05-强制锁定清单.md](05-强制锁定清单.md)。

每次生成、每次改版、每次用户新增参考图或要求换排版时，都必须重新输出：

```text
强制锁定清单：
1. 产品外观锁定：
2. 容量角标锁定：
3. 卖点分区锁定：
4. 证据图锁定：
5. 配件锁定：
6. 版式/价格区锁定：
7. 本次禁止出现：
是否允许进入画面文字预览：等待用户确认
```

容量角标锁定必须明确写：

```text
容量角标只出现一次，属于产品主体右侧上半部信息区。无右侧证据贴片组时，可以靠近产品右侧或产品顶部顶角，但在画面中的垂直位置不得高于指定容量区上沿，不能漂到画面四角成为独立角标。若右侧存在证据贴片组，容量角标必须进入右侧贴片组区域，并且必须作为该组最上方的第一个贴片。容量角标不得出现在左侧标题区、产品下半部、底部腰带、产品正中核心结构上，也不得位于任何证据贴片下方。
```

用户没有确认强制锁定清单前，禁止进入画面文字预览，更禁止生成图片。

### 强制确认门槛 3：画面文字预览确认

生成图片前，必须先把画面中要使用的全部文字内容列出来，并检查重复卖点。

文字预览必须给用户审核。用户没有确认前，禁止生成图片。

### 画面文字清单

按下面格式输出：

```text
顶部品牌背书：
主标题：
副标题/利益解释：
左侧辅助卖点1：
左侧辅助卖点2：
左侧辅助卖点3：
左侧辅助卖点4：
容量角标：
产品证据区文字：
价格区主文案：
价格数字：
价格区权益1：
价格区权益2：
底部服务/售后：
其它角标：
```

### 重复卖点检查

列完文字后，必须检查：

- 主标题卖点是否又出现在辅助卖点里。
- 辅助卖点是否又出现在产品证据区。
- 容量是否只出现一次，并且是否位于产品主体右侧上半部信息区；若有右侧证据贴片组，容量是否作为贴片组最上方第一个贴片。
- 价格权益区是否混入产品功能卖点。
- 右侧贴片已经表达的卖点，左侧是否重复表达。
- 同一权益是否在价格腰带和底部服务条重复出现。

如果发现重复，先删减或合并，再生成图片。

### 用户审核确认

画面文字清单后必须明确询问：

```text
请确认：以上文字是否就是这张图要出现的全部内容？
是否需要补充、删除或调整主次？
确认后我再生成图片。
```

满足下面条件才进入出图：

- 用户已确认信息完整
- 核心卖点只有 1 个
- 辅助卖点 3-5 个
- 价格权益区存在
- 固定素材列表明确
- 未确认参数没有被写成真实数字
- 画面有真实产品约束
- 已完成强制锁定清单并获得用户确认
- 已加入降低 AI 感约束
- 已定义失败记录字段
- 已列出全部画面文字
- 已完成重复卖点检查
- 用户已确认画面文字预览

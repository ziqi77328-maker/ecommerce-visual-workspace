# 强制锁定清单

用于主图生成前、提示词生成前、每次改版前的硬性锁定。只要是电商主图任务，必须先输出本清单并逐项落到提示词里；没有落地的项目不能进入出图。

## 触发时机

每次出现下面任一动作，都必须重新执行强制锁定：

- 第一次生成主图
- 用户新增参考图、产品图、证据图或配件图
- 用户调整版式、背景、腰带、证据区或卖点
- 用户要求“再来一版”“换排版”“更吸引人”“去掉/新增模块”
- 出图失败后重新生成

不能只在第一次生成时锁定，后续每次改版都要重新输出。

## 一、产品真实外观锁定

生成前必须逐项写明：

```text
产品类目：
品牌/型号：
产品主色：
产品材质质感：
产品主体轮廓：
产品比例：
Logo 位置：
控制面板位置：
按键/旋钮/数显：
锅盖/把手/门体/阀门/铰链/通风口：
侧面接口/脚垫/底座：
允许优化：
禁止改变：
```

必须锁定：

- 型号、类目和产品真实结构。
- 产品颜色、材质、主体轮廓、厚度、弧度和比例。
- Logo 的位置和方向；如果用户要求后期 PS Logo，则只预留空位，不额外生成 Logo。
- 控制面板、按键、旋钮、显示屏、菜单文字区域、把手、锅盖、门体、阀门、铰链、通风口、底座、脚垫等关键识别部件。
- 配件、内胆、局部证据图的真实形态和数量。

允许：

- 优化商业摄影光影、清晰度、边缘轮廓光、接触阴影、轻微透视和背景融合。

禁止：

- 换成其它型号。
- 改变产品主色。
- 重新设计产品外观。
- 增减按键、旋钮、面板、把手、阀门、门体、脚垫或接口。
- 拉伸、压扁、裁掉关键结构。
- 让火焰、文字、贴片、证据卡或腰带遮挡控制面板、Logo、锅盖缝、把手、门体和底部轮廓。

## 二、容量角标强制锁定

容量是产品侧判断信息，不是核心卖点。

必须遵守：

- 容量只出现一次。
- 容量角标属于产品主体右侧上半部信息区，不是画面角标。
- 无右侧证据贴片组时，容量角标可以靠近产品右侧或产品顶部顶角，但在画面中的垂直位置不得高于指定容量区上沿，不能漂到画面四角成为独立角标。
- 有右侧证据贴片组时，容量角标必须进入右侧证据贴片组区域，并且必须作为该组最上方的第一个贴片。
- 容量角标不得出现在左侧标题区、产品下半部、底部腰带、产品正中核心结构上，也不得位于任何证据贴片下方。
- 容量不得进入标题、左侧辅助卖点、证据区、价格区、权益区、服务条或底部腰带。

提示词必须写：

```text
容量角标只出现一次，写“【容量】”。容量角标属于产品主体右侧上半部信息区。无右侧证据贴片组时，容量角标可以靠近产品右侧或产品顶部顶角，但在画面中的垂直位置不得高于指定容量区上沿，不能漂到画面四角成为独立角标。若右侧存在证据贴片组，容量角标必须进入右侧贴片组区域，并且必须作为该组最上方的第一个贴片。容量角标不得出现在左侧标题区、产品下半部、底部腰带、产品正中核心结构上，也不得位于任何证据贴片下方。
```

出图后检查：

- 如果容量漂到画面四角成为独立角标，判定失败。
- 如果右侧有证据贴片组但容量不在该组最上方，判定失败。
- 如果容量位于产品下半部、底部腰带或证据贴片下方，判定失败。
- 如果容量进入左侧卖点或标题，判定失败。
- 如果容量重复出现，判定失败。

## 三、卖点分区锁定

必须先锁定每个卖点的唯一表达区。

```text
核心卖点：
核心卖点表达区：
辅助卖点 1：
辅助卖点 1 表达区：
辅助卖点 2：
辅助卖点 2 表达区：
辅助卖点 3：
辅助卖点 3 表达区：
证据区负责证明：
价格权益区只写：
删除/不出现的重复卖点：
```

规则：

- 核心卖点只能在标题区主表达一次。
- 辅助卖点不能复读标题。
- 已在产品贴片、角标、证据卡出现的卖点，不得再进入左侧文字卖点。
- 证据区只证明卖点，不做第二个卖点列表。
- 价格权益区只写成交权益，不写产品功能。

## 四、证据图锁定

每个证据图必须先说明“证明什么”，不能只作为装饰。

```text
证据图 1：
证明卖点：
画面位置：
允许处理：
禁止处理：
```

必须遵守：

- 固定素材优先使用 `常用图片`，保持原始比例、颜色、文字和材质清晰。
- 低糖饭必须展示上方沥糖釜/沥糖篮和下方配套内胆。
- 304、316L、0 涂层、晶钛等材料证据必须使用已确认素材或官方资料，不重新发明。
- 火焰、流光、蒸汽不能遮挡证据核心结构和文字。

## 五、配件锁定

用户提供配件图时，必须锁定：

- 配件数量。
- 每个配件名称。
- 每个配件外观。
- 配件是证据、赠品还是前景实物。
- 配件摆放位置和最大占比。

禁止：

- 把多个配件融合成一个。
- 省略已提供配件。
- 新增未提供配件。
- 让配件替代产品主体。
- 把小证据配件放大成主视觉，除非用户明确要求。

## 六、版式与成交区锁定

每次改版都必须重新判断：

```text
主图目标：
是否需要价格权益：
价格区位置：
价格区是否被用户要求删除：
删除价格区后的主图目标变化：
底部留白用途：
```

如果用户要求去掉价格腰带，必须把主图目标重新切换为功能教育、高端质感或空白后期排版，并明确不再使用活动促销目标。

## 七、每次生成前必须输出

```text
强制锁定清单：
1. 产品外观锁定：
2. 容量角标锁定：
3. 卖点分区锁定：
4. 证据图锁定：
5. 配件锁定：
6. 版式/价格区锁定：
7. 本次禁止出现：
是否允许进入画面文字预览：是/否
```

用户确认后，才能进入画面文字预览；画面文字预览再次确认后，才能生成图片。

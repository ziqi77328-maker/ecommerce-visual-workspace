# AI感降低策略

用于解决生成图片“AI 感太重”、影响电商图片流量的问题。

这里的目标不是完全不用 AI，而是让 AI 只负责背景、光影、版式和氛围，把产品、证据、文字和价格尽量交给真实素材与后期排版控制。

---

## 为什么 AI 感会影响流量

电商主图的点击来自信任感。AI 感太重时，用户会下意识觉得：

- 产品可能不真实
- 材质和功能证据不可信
- 价格和权益像模板广告
- 细节看不清，不敢点进去
- 平台缩略图里信息像噪点

所以主图不能只追求“炫”，要优先像真实商业摄影和真实设计稿。

---

## 最容易产生 AI 感的原因

| 问题 | 表现 |
| --- | --- |
| 背景过度生成 | 梦幻、虚化、粒子太多、空间不合理 |
| 光影不统一 | 产品、内胆、贴片、文字像分别来自不同画面 |
| 产品材质假 | 金属像塑料，玻璃无真实反射，边缘过度锐化 |
| 字效过度 | 红金 3D 大字太厚，像自动生成模板 |
| 证据素材被重绘 | 内胆、沥糖釜、304/316L 贴片文字变糊 |
| 食物不真实 | 米饭、蒸汽、火焰、汤汁像特效 |
| 信息过满 | 每个角落都有贴片，缩略图变脏 |

## 生成策略

### 1. 产品主体优先用真实图

优先级：

```text
官方商品图 > 用户提供实拍图 > 已验证产品素材 > AI 生成产品
```

提示词中必须写：

```text
产品主体来自真实商品图，保持原始型号、比例、颜色、Logo、面板、按钮、把手、锅盖、门体和配件结构；只允许优化商业摄影光影，不重新设计产品。
```

### 2. 背景要像真实摄影棚

推荐写法：

```text
真实商业摄影棚布光，浅色厨房台面或简洁电商摄影背景，自然柔光，轻微环境反射，真实接触阴影，背景清爽不过度虚化，空间透视合理。
```

少用：

```text
梦幻、赛博、超现实、大量粒子、强烈爆闪、漂浮碎片、过度烟雾、夸张流光
```

### 3. 特效只服务卖点

允许：

- 轻微轮廓光
- 柔和金属高光
- 内胆周围局部光圈
- 少量热力或水汽
- 局部功能箭头和证据连接线

避免：

- 大面积火焰包围产品
- 背景满屏粒子
- 产品悬浮无接触阴影
- 特效盖住内胆、贴片和价格

### 4. 文字和价格尽量后期排版

生成图阶段只保留文字安全区和模块位置。最终文字、价格、Logo 和权益最好用设计工具或代码排版叠加。

如果必须在生成图里包含文字，要求：

```text
文字区域清晰、平面印刷设计感、非手写、非乱码、非厚重 3D 生成字，主标题不超过 2 行，价格数字最大。
```

### 5. 固定素材只嵌入，不重绘

对 `常用图片` 的要求：

```text
固定素材保持原始比例、原始颜色和文字清晰度；只允许加真实投影、轻微反光和统一色温，不允许重绘、改字、改颜色或变形。
```

## 提示词加固模块

每次生成都建议加入：

```text
真实电商商业摄影质感，产品为真实商品图合成，非概念设计，非塑料模型，非过度渲染。自然摄影棚柔光，真实接触阴影，材质反射符合产品本身，背景干净，信息层级清楚，手机端缩略图可读。固定证据素材保持原始清晰度和比例，不重绘，不改字，不变形。
```

负面关键词加入：

```text
AI generated look, fake plastic texture, over-rendered, fantasy background, excessive particles, excessive glow, distorted product, wrong logo, garbled text, blurry label, fake 3D text, unrealistic steam, unrealistic fire, floating object without shadow, inconsistent lighting
```

中文负面约束：

```text
不要梦幻背景，不要过度粒子，不要塑料材质，不要厚重生成字，不要重绘产品，不要改变产品结构，不要把固定素材变糊，不要让特效遮挡证据区和价格区。
```

## 生成后检查

出图后按 10 分制判断 AI 感。

| 分数 | 判断 |
| --- | --- |
| 0-2 | 基本真实，可进入后期排版 |
| 3-4 | 有轻微 AI 痕迹，局部修正 |
| 5-6 | 明显生成感，需要重出背景或重做光影 |
| 7-10 | 不建议使用，写入失败数据库 |

检查项：

- 产品边缘是否像真实拍摄
- 产品是否有接触阴影
- 内胆和贴片是否清楚
- 背景是否真实可解释
- 字体是否像设计排版而不是生成字
- 光源方向是否统一
- 缩略图是否干净

---

## 推荐工作方式

更稳的流程：

```text
真实产品图/官方图
+ AI 生成干净背景
+ 固定证据素材嵌入
+ 后期排版文字、价格和权益
+ 缩略图检查
+ 失败数据库复盘
```

不推荐：

```text
让 AI 一次性生成产品、背景、文字、价格、证据贴片和品牌 Logo。
```

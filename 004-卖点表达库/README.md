# 004-卖点表达库

> 用于沉淀电商视觉中高频卖点的表达逻辑。
>
> 记录内容：
>
> - 文案怎么说
> - 视觉怎么表现
> - 特效怎么强化
> - 哪些产品可以复用

---

## 使用方法

每张主图建议采用：

```text
1个核心大卖点 + 3-5个辅助卖点 + 1个产品证据
```

卖点不能在多个区域重复出现。生成前先按 [M-卖点去重与分区规则.md](M-卖点去重与分区规则.md) 分配信息：标题区讲核心卖点，辅助卖点区讲补充理由，容量只做产品角落贴片，证据区用图片证明，价格权益区只讲成交权益。

当卖点命中低糖饭、304 不锈钢、316L 不锈钢、0 涂层、晶钛等已有固定素材时，先调用 [../常用图片/README.md](../常用图片/README.md) 中的素材规则。已有素材优先，不重新生成同类文字贴片。

### 核心大卖点

放在最大标题区，负责第一眼吸引。

示例：

- 22分钟快煮饭
- 316L无缝内胆
- AI雷达煲
- 专业沥糖釜
- 上蒸下煮

### 辅助卖点

放在左侧信息卡、图标列表、功能模块中。

示例：

- 24H安心预约
- 55℃一键保温
- 大功率速沸
- 0涂层不粘
- 9大菜单

### 产品证据

放在产品旁边或右下角局部放大。

示例：

- 内胆局部放大
- 控制面板放大
- 316L盾牌徽章
- 抗菌率晶体徽章
- 容量火焰徽章

容量徽章只能作为产品角落贴片，不进入左侧辅助卖点列表。

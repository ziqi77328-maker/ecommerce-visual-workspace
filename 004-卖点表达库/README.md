> - 文案怎么说

> - 特效怎么强化
> - 哪些产品可以复用

---

## 使用方法

每张主图建议采用：

```text
1个核心大卖点 + 3-5个辅助卖点 + 1个产品证据
```

### 核心大卖点

放在最大标题区，负责第一眼吸引。

示例：

- 22分钟快煮饭
- 316L无缝内胆
- AI雷达煲
- 专业沥糖釜
- 上蒸下煮

### 辅助卖点

放在左侧信息卡、图标列表、功能模块中。

示例：

- 24H安心预约
- 55℃一键保温
- 3L黄金容量
- 大功率速沸
- 0涂层不粘

### 产品证据

放在产品旁边或右下角局部放大。

示例：

- 内胆局部放大
- 控制面板放大
- 316L盾牌徽章
- 抗菌率晶体徽章
- 容量火焰徽章

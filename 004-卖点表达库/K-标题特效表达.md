# K-标题特效表达

> 用于解决主图标题同质化问题。
> 标题特效不要默认都用“红金 3D 大字”，应根据核心卖点选择不同视觉语气。

---

## 使用原则

每张主图先判断核心购买理由，再选择标题特效。

```text
材质卖点 -> 金属质感大字
火力卖点 -> 火力热能大字
科技/屏幕卖点 -> 科技数显大字
健康/清洁卖点 -> 清透健康大字
数字参数 -> 数字辅助视觉
服务权益 -> 权益章辅助表达
```

注意：

- 首发主图优先用“卖点文字主导”。
- 数字主视觉和权益章不作为默认首发方向。
- 同一产品多张图要轮换标题风格，避免一眼同模板。

## 一级常用：卖点文字主导

### 1. 金属质感大字

适合卖点：

- 0 涂层
- 不锈钢内胆
- 精钢厚釜
- 304 不锈钢
- 金属内胆
- 厚釜、钢釜、晶钛类材质

视觉关键词：

- 银色金属字
- 拉丝质感
- 黑色投影
- 白色高光边
- 机械压印感
- 冷光扫过
- 金属切边
- 镜面反射

推荐提示词：

```text
主标题使用银色金属质感大字，拉丝钢纹理，黑色投影，白色高光边，机械压印感，冷光扫过，金属切边，适合表达0涂层不锈钢内胆、精钢厚釜等材质卖点。
```

## 二级备选：特殊视觉主导

### 5. 数字主视觉

适合：

- 360°
- 365 天
- 1 级能效
- 角度、时间、服务天数等强数字

不适合：

- 容量数字。容量只能作为产品主体右侧上半部信息区的小型角标，有右侧证据贴片组时排在该组最上方，不作为标题区或大数字主视觉。

定位：

- 作为第二张图、活动图、参数强化图。
- 不建议作为所有产品首发默认模板。

原因：

- 数字很抢眼，但容易牺牲完整卖点表达。
- 如果首图需要讲清“产品功能 + 材质卖点”，数字应做辅助徽章或局部强化。

推荐提示词：

```text
使用巨型数字作为辅助视觉锚点，数字立体高亮，周围搭配简短卖点解释，不要让数字压过核心产品卖点和产品主体。
```

### 5.1 数字 / 符号型立体锚点

适合：

- 0 涂层
- 0 氟
- 316L
- 1 级能效
- 24H 预约
- 60s 快排
- 99.99% 抗菌率

定位：

- 用作产品旁边的强记忆点。
- 让数字或短符号参与画面空间，而不是只作为平面贴片。
- 适合放在产品、内胆、局部结构、材质盾牌旁边，与证据元素形成一组。

视觉关键词：

- 立体数字
- 侧透视字体
- 金属切边
- 高光描边
- 真实投影
- 台面反射
- 发光百分比
- 上升箭头
- 与产品同透视方向

推荐提示词：

```text
将核心数字或短符号做成立体侧透视视觉锚点，放在产品主体或内胆证据旁边，带金属切边、高光描边、真实投影和轻微反射，与产品保持同一空间透视，不遮挡控制面板、Logo和关键结构。
```

禁忌：

- 不把长句做成立体透视字。
- 不让数字锚点大过产品主体。
- 不让锚点脱离对应证据单独漂浮。
- 不用无法验证的数据做百分比锚点。

---

### 6. 权益章 / 服务章

适合：

- 京仓直发
- 送货上门
- 价保 618
- 365 天只换不修
- 运费险
- 全国联保

定位：

- 作为底部权益条、右上角徽章、辅助信任背书。
- 不作为主标题首发方向。

原因：

- 权益能促成下单，但不是产品核心购买理由。
- 首图第一眼仍应先讲产品卖点。

推荐提示词：

```text
权益章作为辅助信任背书，使用红章、金章、盾牌、官方认证章、红粉胶囊条等形式，放在底部权益区或右上角，不替代主标题。
```

## 微烤炸一体机推荐顺序

首发主图：

```text
标题风格：金属质感大字
主标题：微烤炸一体机
副标题：0涂层不锈钢内胆
```

第二版差异化：

```text
标题风格：火力热能大字
主标题：360°旋风空气炸
副标题：外酥里嫩 少油更香
```

第三版差异化：

```text
标题风格：清透健康大字
主标题：0涂层不锈钢内胆
副标题：少油空气炸 更安心
```

辅助强化：

```text
容量角标：23L容量
权益章：京仓直发 / 价保618
```

## 防同质化组合建议

不要每次都使用：

```text
红金3D标题 + 红金火焰价格腰带
```

推荐错位组合：

```text
金属质感大字 + 红金促销腰带
科技数显大字 + 白红平台补贴区
火力热能大字 + 黑金价格铭牌
清透健康大字 + 白红优惠券价格卡
卖点文字主导 + 权益章作为辅助角标
```

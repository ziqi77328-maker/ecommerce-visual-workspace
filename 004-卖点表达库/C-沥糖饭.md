## 沥糖饭

### 核心规则

- 只要产品定位是低糖电饭煲，画面必须出现沥糖釜。
- 沥糖釜是低糖饭标配证据，不能只用普通内胆或米饭碗替代。
- 推荐做成“上沥糖釜、下配套内胆”的右侧证据组合。
- 有明确还原糖数据时，才加入百分比数据贴片；没有数据时不写降低比例。

### 强制触发规则

当主图文案、用户输入或产品定位里出现下面任一表达时，必须触发沥糖釜证据区：

- 低糖饭
- 沥糖饭
- 控糖饭
- 米汤分离
- 低糖电饭煲
- 专业沥糖釜

触发后，生成关键词和画面描述里必须同时写入：

```text
上方：专业沥糖釜 / 沥糖篮，内盛米饭，圆孔沥水结构清晰
+
下方：产品配套内胆，承接沥糖釜，形成上下组合
+
标签：专业沥糖釜 / 米汤分离
```

不能只写：

```text
316L不锈钢内胆
内胆盛米饭
饭碗
普通配件展示
```

这些只能证明材质或容量，不能证明低糖饭功能。

### 文案表达

- 专业沥糖釜
- 控糖不降香
- 低糖健康饭
- 米汤分离
- 低糖饭标配
- 还原糖降低xx%（仅在产品资料提供数据时使用）

### 图形表达

- 双层沥糖釜
- 米汤分离结构
- 数据标签
- 上方沥糖釜装米饭
- 下方产品配套内胆
- 沥糖釜圆孔沥水结构
- 右侧产品证据区

### 特效表达

- 火焰能量
- 金色流光
- 数据光效
- 蒸汽上升
- 银色金属高光
- 蓝白色数据贴片
- 向下箭头数据徽章

### 视觉强化

- 还原糖数据放大（仅有数据时）
- 双层内胆拆解
- 米汤分离演示
- 沥糖釜与内胆上下叠放
- 数据贴片指向米饭或沥糖釜

### 适用产品

- 电饭煲
- 低糖电饭煲

### 沥糖釜证据区

低糖电饭煲主图里，右侧配件建议做成：

```text
下方：产品配套内胆
+
上方：专业沥糖釜 / 沥糖篮
+
沥糖釜内：米饭
```

表达要点：

- 沥糖釜位于内胆上方，形成明确上下结构。
- 沥糖釜和内胆要有接触阴影、金属反射和空间层次。
- 沥糖釜圆孔、拉丝不锈钢质感、口沿高光要清晰。
- 配件尺寸低于主产品，作为卖点证据，不抢主产品。
- 食物不能遮挡沥糖釜孔位和内胆轮廓。

### 数据贴片表达

当产品资料提供还原糖数据时，可以加入数据可视化贴片：

```text
蓝白色数据徽章
+
向下箭头造型
+
大号百分比数字
+
还原糖文案
```

示例：

- -44.6% 还原糖
- 还原糖降低30.7%

位置建议：

- 悬浮在沥糖釜上方或旁边。
- 箭头指向米饭、沥糖釜或米汤分离区域。
- 贴片不遮挡产品主体、控制面板和配件结构。

禁忌：

- 未提供数据时，不写“-44.6%”“30.7%”等具体数字。
- 不编造检测结果。
- 不让数据贴片替代沥糖釜本体。
- 不用单个普通内胆、单个饭碗或单个米饭配件替代沥糖釜证据区。
- 不把 316L 不锈钢内胆当作低糖饭的唯一证据。
- 不生成“低糖饭”主标题但画面里没有沥糖釜/沥糖篮。

### 生成前自检

生成低糖电饭煲主图关键词前，先检查：

- [ ] 文案是否出现低糖饭、沥糖饭、控糖饭、米汤分离等触发词。
- [ ] 如果出现，画面是否明确包含上方沥糖釜/沥糖篮。
- [ ] 沥糖釜是否有圆孔、口沿、金属高光和米饭颗粒。
- [ ] 下方是否有产品配套内胆，与沥糖釜形成上下组合。
- [ ] 是否没有用单个 316L 内胆替代沥糖釜证据。
- [ ] 未提供还原糖数据时，是否没有写具体百分比。

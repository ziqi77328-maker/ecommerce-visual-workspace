## 沥糖饭

### 核心规则

- 只要产品定位是低糖电饭煲，画面必须出现沥糖釜。
- 低糖电饭煲的沥糖釜证据必须使用 `常用图片/沥糖釜.jpg`，不重新生成、不替换、不变形、不变色。
- 沥糖釜是低糖饭标配证据，不能只用普通内胆或米饭碗替代。
- 大部分情况下优先做成右下角标准证据组合：“上方沥糖釜 + 下方标准电饭煲内胆 + 旁边对应材质贴片”。
- 有明确还原糖数据时，才加入百分比数据贴片；没有数据时不写降低比例。

### 强制触发规则

当主图文案、用户输入或产品定位里出现下面任一表达时，必须触发沥糖釜证据区：

- 低糖饭
- 沥糖饭
- 控糖饭
- 米汤分离
- 低糖电饭煲
- 专业沥糖釜

触发后，生成关键词和画面描述里必须同时写入：

```text
使用常用图片/沥糖釜.jpg 作为固定沥糖釜素材，保持原始比例、原始金属颜色、孔位结构和文字清晰度，不重绘、不变形、不变色、不模糊
+
右下角证据区上方：专业沥糖釜 / 沥糖篮，内盛香软白米饭，圆孔沥水结构清晰
+
右下角证据区下方：用户提供或常用图片中的标准电饭煲内胆，承接沥糖釜，形成上釜下胆的稳定空间关系
+
旁边：对应材质贴片，优先调用常用图片中的 304 / 316L / 晶钛 / 0涂层素材
+
特效：内胆底部加金色光圈、轻微火焰热力、环绕流光和真实接触阴影
```

不能只写：

```text
316L不锈钢内胆
内胆盛米饭
饭碗
普通配件展示
```

这些只能证明材质或容量，不能证明低糖饭功能。

### 文案表达

- 低糖健康饭
- 米汤分离
- 低糖饭标配
- 还原糖降低xx%（仅在产品资料提供数据时使用）

### 图形表达

- 双层沥糖釜
- 米汤分离结构
- 数据标签
- 上方沥糖釜装米饭
- 下方产品配套内胆
- 沥糖釜圆孔沥水结构
- 右侧产品证据区

### 特效表达

- 火焰能量
- 金色流光
- 数据光效
- 蒸汽上升
- 银色金属高光
- 蓝白色数据贴片

- 电饭煲

### 沥糖釜证据区

低糖电饭煲主图里，右下角配件优先做成标准证据组合：

```text
上方：常用图片/沥糖釜.jpg
沥糖釜内：香香软软的白米饭
+
下方：用户提供或常用图片中的标准电饭煲内胆
+
旁边：对应材质贴片
```

表达要点：

- 沥糖釜位于内胆上方，形成明确“上釜下胆”的上下结构。
- 沥糖釜内必须装白米饭，米饭颗粒饱满、柔软、有热气和食欲感。
- 下方内胆是标准电饭煲内胆，不用饭碗、汤碗、盘子或无关锅具替代。
- 材质贴片放在内胆旁边或左下侧，靠近内胆但不遮挡内胆主体。
- 如果材质是 316L，优先调用 `常用图片/316母婴级不锈钢材质贴片1.png` 或 `常用图片/316母婴级不锈钢材质贴片2.png`。
- 如果材质是 304，优先调用 `常用图片/304不锈钢材质贴片1.png` 或 `常用图片/304不锈钢材质贴片2.jpg`。
- 沥糖釜和内胆要有接触阴影、金属反射和空间层次。
- 沥糖釜圆孔、拉丝不锈钢质感、口沿高光要清晰。
- 内胆底部可以加金色光圈、热力火焰、环绕流光和轻微蒸汽，强化主图促销质感。
- 配件尺寸低于主产品，作为卖点证据，不抢主产品。
- 食物不能遮挡沥糖釜孔位和内胆轮廓。

### 标准构图优先级

优先级从高到低：

1. 右下角标准组合：上方沥糖釜装白米饭，下方标准电饭煲内胆，旁边材质贴片。
2. 右侧中下部组合：产品主体较大时，可以把证据组放在产品右侧中下部，但仍保持上釜下胆。
3. 独立局部放大组合：空间不足时，可以做成小型局部证据框，但仍必须有沥糖釜、白米饭、下方内胆和材质贴片。

不要把沥糖釜单独漂浮在画面中，也不要只放一个内胆或一个饭碗来表达低糖饭。

### 数据贴片表达

当产品资料提供还原糖数据时，可以加入数据可视化贴片：

```text
蓝白色数据徽章
+
向下箭头造型
+
大号百分比数字
+
还原糖文案
```

示例：

- -44.6% 还原糖
- 还原糖降低30.7%

位置建议：

- 悬浮在沥糖釜上方或旁边。
- 箭头指向米饭、沥糖釜或米汤分离区域。
- 贴片不遮挡产品主体、控制面板和配件结构。

禁忌：

- 不重新绘制沥糖釜，不把 `常用图片/沥糖釜.jpg` 改色、拉伸、压扁或模糊化。
- 未提供数据时，不写“-44.6%”“30.7%”等具体数字。
- 不编造检测结果。
- 不让数据贴片替代沥糖釜本体。
- 不用单个普通内胆、单个饭碗或单个米饭配件替代沥糖釜证据区。
- 不把沥糖釜和内胆左右并排放成两个孤立配件；优先保持上釜下胆关系。
- 不让材质贴片替代沥糖釜和内胆组合。
- 不把 316L 不锈钢内胆当作低糖饭的唯一证据。
- 不生成“低糖饭”主标题但画面里没有沥糖釜/沥糖篮。

### 生成前自检

生成低糖电饭煲主图关键词前，先检查：

- [ ] 文案是否出现低糖饭、沥糖饭、控糖饭、米汤分离等触发词。
- [ ] 如果出现，画面是否使用 `常用图片/沥糖釜.jpg` 作为固定沥糖釜素材。
- [ ] 沥糖釜是否没有变形、变色、文字模糊或孔位丢失。
- [ ] 沥糖釜是否有圆孔、口沿、金属高光和米饭颗粒。
- [ ] 沥糖釜里是否装有香软白米饭，米饭没有遮挡关键孔位。
- [ ] 下方是否有产品配套内胆，与沥糖釜形成右下角上釜下胆组合。
- [ ] 旁边是否放置对应材质贴片，且贴片来自 `常用图片` 文件夹。
- [ ] 内胆周围是否有适度光圈、火焰热力、流光或真实接触阴影，增强主图质感。
- [ ] 是否没有用单个 316L 内胆替代沥糖釜证据。
- [ ] 未提供还原糖数据时，是否没有写具体百分比。

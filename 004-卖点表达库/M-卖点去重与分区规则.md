# M-卖点去重与分区规则

> 用于解决生图时同一个卖点在标题、贴片、证据区、价格区里反复出现，导致画面拥挤、重复、低质感的问题。

---

## 核心原则

同一个卖点只能有一个主表达区，其它区域只能做证据补充，不能原文复读。

```text
标题区：只放核心大卖点
辅助卖点区：放 3 个左右不同类型的小卖点，不放容量
容量角标区：只放产品主体右侧上半部信息区的容量标识；如有右侧证据贴片组，容量必须是该组最上方第一个贴片
产品证据区：用实物、局部、结构或场景证明卖点
价格权益区：只放成交权益，不放产品功能卖点
品牌背书区：只放品牌、官方、售后或平台信任
```

---

## 卖点分区规则

### 1. 标题区

标题区只承担一个任务：让用户第一眼知道主卖点是什么。

适合放：

- 微烤炸一体机
- 0涂层不锈钢内胆
- 20L大平板微波炉
- 22分钟快煮饭

不适合：

- 把 3-5 个辅助卖点都塞进标题
- 标题写一次，贴片再写一次，价格区再写一次

### 2. 辅助卖点区

辅助卖点区负责补充购买理由，建议控制在 3 个左右。

要求：

- 每个辅助卖点必须和标题区不同。
- 每个辅助卖点只出现一次。
- 辅助卖点之间不能表达同一件事。
- 容量不进入辅助卖点区。
- 右侧贴片、产品角标、局部证据卡已经出现的卖点，不能再进入辅助卖点区。

示例：

```text
标题：微烤炸一体机
产品主体右侧上半部容量角标：23L容量
辅助卖点：一级能效 / 360°旋风空气炸 / 0涂层内胆证据
```

### 2.1 容量表达强制规则

容量不可以出现在文字并列卖点中，只能以独立贴片形式放在产品主体右侧上半部信息区。无右侧证据贴片组时，容量角标可以靠近产品右侧或产品顶部顶角，但在画面中的垂直位置不得高于指定容量区上沿，不能漂到画面四角成为独立角标。有右侧证据贴片组时，容量角标必须进入右侧证据贴片组区域，并且必须作为该组最上方的第一个贴片。

正确：

- 产品主体右侧上半部容量角标：5L 大容量
- 右侧证据贴片组最上方角标：4L 容量
- 容量角标靠近产品主体，不进入左侧卖点卡

错误：

- 左侧卖点列表写“4L容量”
- 标题区写容量，右侧又出现容量角标
- 容量角标出现后，左侧再写“满足一家4-6人”
- 容量同时出现在标题、左侧卖点、右侧贴片、价格区
- 容量角标漂到画面四角成为独立角标
- 右侧有证据贴片组时，容量角标放在证据贴片下方
- 容量角标放在产品下半部、底部腰带或产品正中核心结构上

提示词必须写：

```text
容量只出现一次，作为产品主体右侧上半部信息区的小型角标；无右侧证据贴片组时可靠近产品右侧或产品顶部顶角，但不能高过指定容量区上沿、不能漂到画面四角；有右侧证据贴片组时必须进入右侧贴片组区域，并作为该组最上方第一个贴片；不进入左侧文字卖点列表；画面中除容量角标外，不再出现任何容量相关文字。
```

### 2.2 右侧贴片与左侧卖点互斥规则

只要某个卖点已经出现在右侧贴片、产品角标、局部证据卡中，就绝对不能再出现在左侧文字卖点列表里。

正确：

```text
右侧贴片：2级能效
左侧卖点：1300W猛火 / 4种口感 / 配不锈钢蒸笼
```

错误：

```text
右侧贴片：2级能效
左侧卖点：2级能效 省电节能
```

提示词必须写：

```text
右侧贴片出现过的卖点，左侧文字卖点区不得重复。左侧只放未在右侧贴片中出现的不同卖点。
```

### 3. 产品证据区

证据区要用图片解释卖点，不能变成第二个文案区。

适合：

- 开门不锈钢内胆图证明 0涂层不锈钢内胆
- 平板底盘清洁场景证明大平板易清洁
- 双旋钮局部放大证明机械旋钮操控
- 沥糖釜 + 配套内胆证明低糖饭

不适合：

- 证据图旁边再次大字复读标题
- 同一个参数同时做贴片、徽章和大标题
- 多个证据模块都在讲同一个卖点

### 4. 价格权益区

价格权益区只负责成交，不重复产品功能。

适合：

- 活动到手价
- 京仓直发
- 送货上门
- 价保
- 正品保障
- 全国联保
- 赠品 / 返现 / 补贴

不适合：

- 23L大容量
- 0涂层不锈钢内胆
- 一级能效
- 360°空气炸
- 大平板易清洁

## 生成提示词写法

在提示词里必须明确写：

```text
全图卖点不重复：标题区、辅助卖点区、产品证据区、价格权益区各自承担不同信息。
同一个卖点不要在多个位置重复出现。
容量只出现一次，作为产品主体右侧上半部信息区的小型角标；无右侧证据贴片组时可靠近产品右侧或产品顶部顶角，但不能高过指定容量区上沿、不能漂到画面四角；有右侧证据贴片组时必须进入右侧贴片组区域，并作为该组最上方第一个贴片；不进入左侧文字卖点列表；画面中除容量角标外，不再出现任何容量相关文字。
右侧贴片出现过的卖点，左侧文字卖点区不得重复。
证据区只用图片证明卖点，不重复大标题文案。
价格权益区只写成交权益，不写产品功能卖点。
价格腰带主权益和下方权益贴片不得重复。每个权益只出现一次。
```

---

## 出图后检查

- [ ] 核心卖点只在标题区主表达一次。
- [ ] 辅助卖点没有和核心标题重复。
- [ ] 每个辅助卖点只出现一次。
- [ ] 容量只出现一次，位于产品主体右侧上半部信息区；如果有右侧证据贴片组，容量是该组最上方第一个贴片；没有进入标题、左侧卖点、证据区或价格区。
- [ ] 右侧贴片已经出现的卖点，没有再出现在左侧文字卖点区。
- [ ] 产品证据区是在证明卖点，不是在复读卖点。
- [ ] 价格权益区没有出现产品功能卖点。
- [ ] 价格腰带主权益和下方小贴片没有重复表达同一件事。
- [ ] 品牌背书区没有被拿来重复写产品卖点。
- [ ] 删除重复卖点后，价格、权益和产品证据仍然完整。

## 常见错误

### 错误 1：同一个卖点四处出现

```text
标题：0涂层不锈钢内胆
左侧卡片：0涂层不锈钢内胆
内胆证据旁：0涂层不锈钢内胆
底部权益：0涂层不锈钢内胆
```

更好：

```text
标题：0涂层不锈钢内胆
证据区：开门内胆实物图 + 细箭头指向金属内胆
底部权益：京仓直发 / 价保30天 / 正品保障
```

### 错误 2：容量进入左侧卖点

```text
左侧标题区：4L容量
左侧卖点：4L容量 / 满足一家4-6人
```

更好：

```text
产品主体右侧上半部信息区：4L容量
左侧卖点：1300W猛火 / 4种口感 / 配不锈钢蒸笼
```

容量只保留产品主体右侧上半部信息区的小型角标；有右侧证据贴片组时排在该组最上方，不进入文字并列卖点。

### 错误 3：证据区变成第二个卖点列表

```text
内胆图旁边堆：0涂层 / 不锈钢 / 易清洁 / 安心用 / 耐高温
```

更好：

```text
内胆图旁边只保留 1 个短标签：不锈钢内胆证据
其它解释放回辅助卖点区或删掉。
```

### 错误 4：价格腰带权益重复

```text
主权益：今日前10名下单送价值209元电磁炉
小贴片：限量赠品 / 前10名专享 / 下单即赠 / 赠完即止
```

这些都在重复“前几名下单送赠品”。

更好：

```text
方案 A：
主权益：送价值209元电磁炉
小贴片：前10名专享 / 随单发出 / 咨询客服 / 库存有限

方案 B：
主权益：今日前10名下单专享
小贴片：赠电磁炉 / 价保30天 / 京仓速发 / 正品保障
```

主权益负责最大成交刺激，小贴片只补充不同服务或限制条件。

## 内胆证据标准构图

用于电饭煲、电压力锅等主图中的内胆、沥糖釜和材质贴片组合。

### 低糖电饭煲标准组合

大部分低糖电饭煲主图，右下角优先使用下面关系：

```text
上方：常用图片/沥糖釜.jpg
内容：沥糖釜里装香香软软的白米饭
+
下方：用户提供或常用图片中的标准电饭煲内胆
+
旁边：对应材质贴片
```

要求：

- 上方沥糖釜不能变形、变色、重绘或文字模糊。
- 下方内胆必须是标准电饭煲内胆，不能用饭碗、汤碗、盘子或普通锅替代。
- 沥糖釜和内胆要有上下承接关系、真实接触阴影和金属反射。
- 材质贴片放在内胆旁边或左下侧，不能遮挡内胆和米饭。
- 组合整体放在右下角或产品右侧中下部，尺寸低于产品主体，不抢主产品。

### 晶钛内胆标准表现

命中晶钛内胆时，优先使用 `常用图片/晶钛电饭煲内胆1.png`。

```text
晶钛内胆作为产品旁边或右下角证据图
+
内胆内可盛白米饭，口沿和晶钛标识清晰
+
顶部或旁边放置晶钛 / 0涂层材质贴片
+
底部加入金色光圈、环绕流光、热力火焰和真实投影
```

注意：

- 不把晶钛内胆画成普通不锈钢碗。
- 不改变内胆原始颜色和标识。
- 不让火焰遮挡晶钛标识、口沿或米饭。

### 0涂层内胆标准表现

命中 0 涂层电饭煲内胆时，优先使用：

- `常用图片/0涂层电饭煲内胆1.png`
- `常用图片/0涂层电饭煲内胆 特效.png`

```text
0涂层内胆作为独立证据图
+
内胆完整露出口沿、内壁和材质细节
+
旁边放 0涂层材质贴片
+
底部加入金色光圈、橙色热力火焰、环绕流光和轻微蒸汽
```

注意：

- 不能只用“0涂层”文字表达。
- 不能把内胆拉伸成椭圆或压扁。
- 不能把内胆颜色改成金色、黑色或其它不真实颜色。

### 材质贴片搭配

- 304 不锈钢：使用 `常用图片/304不锈钢材质贴片1.png` 或 `常用图片/304不锈钢材质贴片2.jpg`。
- 316L/316 母婴级不锈钢：使用 `常用图片/316母婴级不锈钢材质贴片1.png` 或 `常用图片/316母婴级不锈钢材质贴片2.png`。
- 0 涂层、晶钛：优先使用常用图片中的内胆素材和对应贴片表达。

材质贴片只作为证据辅助，不进入左侧文字卖点重复表达。

### 生成前固定提示

```text
内胆证据区使用常用图片文件夹中的固定素材，保持原始比例、原始颜色、材质标识和文字清晰度。低糖饭优先采用右下角上沥糖釜、下标准电饭煲内胆、旁边材质贴片的组合；晶钛和0涂层内胆加入金色光圈、环绕流光、热力火焰和真实投影，但不遮挡内胆标识、口沿和米饭。
```

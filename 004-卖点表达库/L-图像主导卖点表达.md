# 图像主导卖点表达

## 表达定位

这是一种和“左文右品信息卡”“卖点标签列表”并列的主图卖点表达形式。

核心不是让图片辅助文字，而是让文字辅助图片：

```text
先用一张有冲击力的功能证据图吸引用户
+
再用数字、短词、箭头和小模块解释这张图证明了什么
```

适合需要避免主图同质化、增强视觉吸引力、提升卖点可信度的功能型产品。

---

## 核心原则

### 1. 图片是主角，文字是解释

传统表达容易变成：

```text
大标题 + 多条卖点 + 产品图作为辅助证明
```

图像主导表达应该变成：

```text
功能证据图先抓眼
+ 
文字和数字贴着图解释卖点
```

用户第一眼先被画面吸引，第二眼再通过文字理解卖点具体是什么。

---

### 2. 不依赖整块背景板，也能形成完整阅读区

卖点区不一定要用一整块大卡片或大背景板承载。

可以用多个小模块围绕同一张功能证据图集中排布：

- 大标题
- 数据条
- 功能短条
- 结果数字
- 箭头
- 局部结构图
- 立体关键词
- 光效和动线

这些模块靠位置、方向、颜色和内容关系形成一个视觉组团，让用户自然把它们读成同一个卖点区域。

这样做的好处：

- 画面更融合，不像左侧贴了一块 PPT 信息板。
- 背景、产品、卖点之间更协调。
- 信息集中但不沉闷。
- 可减少同质化的信息卡排版。

---

### 3. 所有模块围绕同一个卖点互相解释

图像主导表达不能把多个功能点随意堆在一起。

所有文字、数字和小图都要解释同一个核心卖点：

```text
核心结果
为什么成立
怎么做到
效果有多明显
用什么结构证明
```

示例结构：

```text
低脂浓香
+ 92.8%祛脂率
+ 60s自动快排
+ 祛脂釜去脂增浓
+ 浓郁度 / 香味 / 蛋白质提升数据
+ 凝鲜阀大沸腾过程图
```

这不是分散卖点，而是一套围绕“低脂浓香”的完整解释。

---

### 4. 只学习表达手法，不照搬他人品牌结构

学习竞品或优秀案例时，只提取卖点组织方式、阅读路径和视觉证据关系。

不能直接照搬他人的品牌背书区、品牌色、活动口号、权益图形和专属标识。

品牌背书要使用本品牌已有规范。例如美的家电主图常用：

- 左上角蓝色品牌背书区
- 白色 Midea 标识
- 蓝白权益徽章，如“365天只换不修”
- 品牌区只占左上角，不压住产品主体和功能证据图

表达重点是：

```text
学习“卖点怎么被图片解释”
不学习“别人品牌怎么露出”
```

---

### 5. 数据必须来自产品资料，不能复制参考图数据

参考图里的数据泡泡、百分比、提升比例只能学习呈现方式，不能直接挪用。

如果产品资料只提供：

```text
240次压力加减
120次沸腾翻滚
```

画面就只能围绕这两个数据组织，不应自动生成类似“29.2%提升”“38.1%提升”“325%提升”等无依据数据。

数据表达规则：

- 有真实数据时，数据贴近它证明的结构或过程。
- 无真实数据时，用“结构名 + 过程描述 + 用户结果”替代。
- 不用参考图中的数字填空。
- 不让 AI 为了画面热闹自动补百分比。

---

## 画面组织方式

### 推荐结构

```text
核心标题
+
一张功能证据图 / 结构过程图
+
贴近图片的数据数字
+
指向图片的箭头或光效
+
解释图片的短文字
+
立体数字或立体关键词强化记忆点
```

重点是让卖点长在图里，而不是贴在图外。

---

### 阅读顺序

建议控制成 4 步：

```text
第 1 眼：看到有冲击力的功能图或立体数字
第 2 眼：看到核心标题，知道卖点方向
第 3 眼：看到数据和箭头，理解效果强度
第 4 眼：看到结构/过程说明，相信卖点来源
```

---

## 可复用表达公式

### 公式 1：功能图 + 数据解释

```text
【功能证据图】 + 【数据】 + 【结果短词】
```

示例：

```text
沥糖釜米汤分离图 + 还原糖降低数据 + 低糖不减香
```

适合：

- 低糖饭
- 祛脂饭
- 大火力
- 快排
- 沸腾
- 蒸烤炸循环热风

---

### 公式 2：结构图 + 机制解释

```text
【局部结构图】 + 【动作箭头】 + 【机制短句】
```

示例：

```text
凝鲜阀结构图 + 蒸汽/油脂排出箭头 + 60s自动快排
```

适合：

- 沥糖釜
- 厚釜内胆
- 蒸汽阀
- 双胆分区
- 热风循环
- 顶部加热管

---

### 公式 3：立体数字 + 材质证据

```text
【立体数字/关键词】 + 【产品局部】 + 【安心短句】
```

示例：

```text
立体 0 涂层 + 不锈钢内胆 + 入口更安心
```

适合：

- 0涂层
- 0氟
- 0添加
- 316L
- 304
- 3L / 4L / 5L
- 22min
- 24H

## 立体关键词表达

产品旁边可以把核心数字或关键词做成立体侧透字，让文字成为画面物体。

### 推荐做法

- 数字或关键词带厚度。
- 字体有轻微侧面透视。
- 放在产品、内胆、食材旁边，像真实立在空间里。
- 颜色与产品材质协调，常用白色、银色、金色。
- 加轻微投影、边缘高光和接触关系。
- 不遮挡产品主体、控制面板和关键结构。

### 可用对象

```text
0涂层
0氟
316L
304
3L
4L
5L
22min
24H
```

### 作用

- 把卖点从平面标签变成视觉记忆点。
- 增强画面空间感。
- 让核心数字比普通文字更容易被扫到。
- 避免所有卖点都变成相似的信息条。

## 材质盾牌表达

当卖点涉及材质安全、食品接触、母婴级、316L、304、0涂层等信任型信息时，建议使用金属盾牌徽章承接。

### 适合表达

```text
316L 母婴级 不锈钢
304 食品接触级
0涂层
0氟
耐腐蚀
入口安心
```

### 推荐做法

- 盾牌放在对应材质证据旁边，例如 316L 不锈钢内胆旁。
- 使用银色金属浮雕、铆钉、边缘高光，表现可信和耐用。
- 盾牌尺寸低于产品主体，不抢产品。
- 盾牌文字要短，优先放核心材质和信任词。
- 盾牌和内胆有空间关系，不要孤立漂浮。

### 不推荐

- 只把“316L”写成普通标签。
- 盾牌离内胆太远，用户不知道它证明什么。
- 盾牌过大，变成主视觉。
- 同一画面堆多个盾牌，造成廉价感。

## 0氟0涂层双0立体证据组

适合电饭煲、电压力锅、微烤炸一体机等有内胆材质卖点的产品。

核心不是把“0氟0涂层”写成一行平面标签，而是把两个“0”做成产品旁边的空间锚点，让用户先记住健康安全，再通过内胆和材质盾牌相信这个卖点。

### 结构公式

```text
白色/银色立体 0
+
短词：0涂层 / 0氟
+
对应实物内胆
+
316L / 304 / 母婴级金属盾牌
+
少量分子粒子、金色光点或清透流光
```

### 可复用关键词

```text
0氟0涂层双0立体证据组、白色银边立体0字、银色金属侧透视数字、产品旁边的空间锚点、内胆旁立体0涂层、内胆旁立体0氟、双0健康记忆点、两个0分别贴近对应内胆、0字与内胆同一透视方向、金属切边、高光描边、真实接触阴影、轻微台面反射、白银色材质、清透健康光效、少量分子粒子、金色光点、内胆实物证据、锅口和内壁清晰、材质反光真实、316L金属盾牌、304食品接触级盾牌、母婴级不锈钢盾牌、盾牌靠近内胆、不遮挡产品主体、不遮挡控制面板、不遮挡内胆口沿
```

### 生成提示词片段

```text
在产品右下角或产品旁边设置“0氟0涂层双0立体证据组”。放置真实内胆证据，内胆完整露出口沿、内壁和材质反光。内胆旁放白色银边立体“0”字，文字为“0涂层”；另一个对应内胆旁放较小的白色银边立体“0”字，文字为“0氟”。两个0字有侧透视、金属切边、高光描边、真实接触阴影和轻微台面反射，像真实立在内胆旁边。旁边加入银色金属盾牌，文字只写官方确认的材质，例如“316L 母婴级 不锈钢”或“304 食品接触级”。整体作为材质健康证据区，尺寸低于产品主体，不遮挡Logo、控制面板、锅口、米饭和关键结构。
```

### 使用限制

- 0氟、0涂层、316L、304、母婴级等材质词必须来自官方资料或用户确认。
- 如果没有确认材质编号，只保留“0涂层”或“健康内胆”等已确认信息，不自动生成 316L / 304。
- 立体 0 不能大过产品主体，不能脱离内胆或材质证据单独漂浮。
- 不把“0氟0涂层316L不锈钢内胆”整句做成立体字，立体部分只保留数字或极短关键词。
- 参考图中的百分比、提升率、祛脂率只能学习数据条形式，不能复制具体数值。

## 品牌背书区与成交区

图像主导表达不等于弱化品牌和价格，但品牌、价格、权益要各自承担清晰职责。

### 品牌背书区

品牌背书适合放在左上角，尺寸克制，形成第一眼可信来源。

美的主图可使用：

```text
左上角蓝色品牌区
+ 白色 Midea 标识
+ 365天只换不修徽章
```

注意：

- 不用竞品或其他案例里的红色官方正品条。
- 不让品牌区变成横贯顶部的大红条，避免压住画面。
- 品牌区只负责背书，不承担核心卖点解释。

### 成交价格区

价格适合放在底部或左下角，和卖点证据区分开。

示例：

```text
活动到手价 699
+ 全国联保
+ 极速发货
+ 365天以换代修
```

价格区负责推动下单，不能抢走功能证据图的第一视觉。

## 电压力锅案例应用

当产品为 6L 美的电压力锅，卖点包括：

```text
0氟0涂层，双胆双模式
免排气
0氟珐琅圆釜
一锅双胆
专利祛脂釜
6L
活动到手价699
```

推荐用图像主导表达组织为：

```text
左上：蓝色 Midea 品牌背书区 + 365天只换不修
中右：产品主体，保持真实比例、面板和品牌清晰
左下/中左：专利祛脂釜功能证据图，作为第一视觉吸引点
功能图周围：240次压力加减、120次沸腾翻滚、免排气
右下：双胆实物组合，表达一锅双胆
内胆旁：316L 母婴级不锈钢盾牌
产品/内胆旁：0涂层、0氟立体侧透字
底部：活动到手价699 + 售后/发货权益
```

表达逻辑：

```text
先看祛脂釜功能图
再看数据解释工作过程
再看双胆和盾牌相信材质
最后看价格和权益完成下单理由
```

### 标题选择逻辑

这类主图的主标题要讲用户第一眼能理解的购买理由，而不是把所有参数堆进标题。

对这个案例来说，主标题推荐：

```text
0氟0涂层，双胆双模式
```

原因：

- “0氟0涂层”连续两个 0，第一眼更容易形成安全、健康、无负担的记忆点。
- “双胆双模式”能直接对应画面里的两个内胆，让标题和图片互相解释。
- 标题从单一材质配置升级为“健康材质 + 双胆使用场景”的组合卖点。
- 316L 仍然重要，但更适合下沉到银色盾牌里作为材质证据。

不建议把主标题写成：

```text
0涂层316L不锈钢内胆
```

这个表达更像材质参数说明，会弱化“0氟珐琅圆釜”和“一锅双胆”的画面证据。

更清晰的层级是：

```text
主标题：0氟0涂层，双胆双模式
盾牌证据：316L 母婴级 不锈钢
实物证据：316L不锈钢内胆 + 0氟珐琅圆釜
```

## 与信息卡表达的区别

| 表达方式 | 第一眼吸引物 | 文字作用 | 风险 |
| --- | --- | --- | --- |
| 信息卡表达 | 大标题 / 卡片 / 标签 | 主体信息承载 | 容易像模板，画面割裂 |
| 图像主导表达 | 功能图 / 结构图 / 立体数字 | 解释图片和数据 | 图像证据弱时会看不懂 |

两种方式可以并列使用。

当产品缺少强功能画面，使用信息卡表达更稳。

当产品有清楚的结构、过程、材质或数据证据时，优先考虑图像主导表达。

## 自检清单

- [ ] 第一眼吸引用户的是图片、结构图、过程图或立体数字，而不是单纯文字。
- [ ] 文字和数字是在解释图片，不是图片在陪衬文字。
- [ ] 卖点模块没有依赖一整块大背景板。
- [ ] 小模块集中在同一个视觉阅读区里。
- [ ] 所有信息都围绕同一个核心卖点互相解释。
- [ ] 数据贴近它所证明的结构、过程或结果。
- [ ] 箭头、光效、局部图有明确指向关系。
- [ ] 立体数字或立体关键词没有遮挡产品主体。
- [ ] 品牌背书使用本品牌规范，没有照搬参考图品牌条。
- [ ] 316L、0涂层等信任型材质卖点有盾牌或内胆实物承接。
- [ ] 没有出现产品资料之外的百分比、提升率和实验数据。
- [ ] 画面看起来是一张完整商业图，而不是产品图上贴了很多标签。

# E-价格视觉风格

> 用于解决价格区同质化问题。
> 价格区不要每张都默认“红金火焰底栏”，应根据产品调性、促销强度和画面风格选择。

---

## 使用原则

价格区首先要清楚，其次才是特效。

```text
强促销 -> 红金促销腰带
清透高光 -> 白红平台补贴区
高端金属 -> 黑金价格铭牌
领券返现 -> 优惠券价格卡
秒杀直降 -> 闪电直降条
```

注意：

- 价格数字必须是价格区最大元素。
- 右侧优先放权益，不要重复小卖点。
- 价格区不能遮挡产品主体和产品证据。
- 主价格腰带高度控制在画面高度 14%-18%；含底部服务条时，总成交区高度不超过 20%-22%。
- 价格腰带主权益和下方小贴片不得重复，每个权益只出现一次。
- 同一系列多张图要轮换价格视觉，避免同质化。
- 火焰不是必要视觉元素，只在强促销且不影响产品主体时使用。

## 一级常用：主图首发推荐

### 1. 红金促销腰带

适合：

- 618、双 11、平台大促
- 到手价
- 限时活动
- 强低价冲击

视觉关键词：

- 底部红金横条
- 火焰底托（可选）
- 金色描边
- 超大价格数字
- 白黄渐变数字
- 红金爆闪
- 速度线
- 流光

推荐结构：

```text
左侧活动标签
+
中间超大价格
+
右侧权益条
```

示例：

```text
活动到手价
¥799
京仓直发 送货上门 / 价保618 早买早享受
```

注意：

- 右侧放权益，不要重复“23L、0涂层、空气炸”等小卖点。
- 容量不放入价格区，不放入左侧卖点，只能作为产品主体右侧上半部信息区的小型角标；有右侧证据贴片组时必须排在该组最上方。
- 适合强促销，但不能每张都用。
- 火焰可以替换为流光、爆闪、描边、速度线或金属高光，不能把火焰当成默认必选项。

### 1.1 亮晶晶厚实红金腰带

适合：

- 产品主体已经成立，但画面缺少电商精修质感。
- 促销信息需要更“厚”、更亮、更有点击冲击力。
- 黑色家电放在浅色背景中，需要底部形成强成交锚点。

视觉关键词：

- 厚实红金立体促销腰带
- 深红底板
- 金色金属倒角
- 斜切金属边
- 玻璃高光扫光
- 红金流光丝带
- 细小金色粒子
- 星芒高光
- 轻微外发光
- 价格数字白金填充
- 红色阴影
- 金色描边

推荐结构：

```text
左侧活动到手价
+
中间超大立体价格数字
+
右侧胶囊权益徽章
```

提示词：

```text
底部做厚实亮晶晶红金立体促销腰带，高度控制在画面 14%-18%，不遮挡产品。若有下层服务权益条，总成交区高度不超过 22%。腰带有深红底板、金色金属倒角、斜切边、玻璃高光扫光、红金流光丝带、细小金色粒子和星芒。主价格数字使用白金填充、红色阴影、金色描边和轻微外发光。权益做整齐胶囊徽章，带金色边框和微弱高光，且不与主权益重复。
```

注意：

- 亮晶晶是质感增强，不是铺满火焰。
- 主价格腰带不能超过 18% 高度；含下层服务条的总成交区不能超过 22%。
- 特效不能遮挡产品、文字、价格和产品证据。
- 权益胶囊数量建议 3-4 个，过多会显得乱。
- 主权益和权益胶囊不能重复表达同一件事。
- 产品仍应是第一视觉中心，价格区是成交收口。

### 1.2 双层大促腰带 + 赠品压边

适合：

- 价格很强，同时赠品价值也需要第一眼看见。
- 618、双 11、价保、晒单返现、赠品同场表达。
- 产品主体在中右侧，底部可以承接成交信息。

视觉关键词：

- 底部高饱和红粉主价格腰带
- 左侧超大白色价格数字
- 价格下方白色圆角胶囊写活动到手价
- 右侧大字赠品价值
- 黄色金额强调
- 下方浅粉服务保障胶囊（可选）
- 价保 618
- 早买早享受
- 赠品实物压在腰带上沿
- 赠字红色小火焰角标
- 晒单返现浮动在腰带上方

推荐结构：

```text
上层成交腰带：
左侧超大到手价 + 中间赠品实物 + 右侧赠品价值文案

下层服务胶囊：
价保618 + 早买早享受 + 平台情绪图标

腰带上方辅助权益：
晒单返现金额，贴近赠品和产品，不进入主价格数字区
```

提示词：

```text
底部做双层红粉大促价格腰带。上层为高饱和玫红底板，左侧放超大白色价格数字“¥【真实价格】”，数字占上层腰带高度 70% 左右，价格下方用白色圆角胶囊写“活动到手价”。中间放赠品实物小图，赠品从腰带上沿向上压出，旁边加红色“赠”火焰角标。右侧写“送价值【真实金额】元美的养生壶”，其中金额用亮黄色放大。下层为浅粉色长胶囊服务保障条，写“价保618 早买早享受”，带轻微高光和红色强调字。晒单返现作为腰带上方浮动权益，靠近赠品与产品下方，不能压住主价格。
```

简化提示词：

```text
如果底部信息过厚或产品空间被挤压，把价格腰带整体下移贴近底边，并去掉最底部服务保障条，只保留上层主价格腰带、赠品实物、赠品价值文案和必要的限量赠品胶囊。价保、京仓速发、当日达等服务权益不再单独做最底部一行，避免底部过满。
```

注意：

- 赠品实物要贴近价格腰带，形成“买这个价格还能拿到赠品”的成交关系。
- 赠品不要遮挡价格数字，也不要离价格区太远。
- 赠品文案与赠品实物必须相邻，不能一个在左一个在右完全断开。
- 晒单返现适合放在腰带上方，作为进入价格腰带前的额外刺激。
- 下层服务胶囊只放价保、早买早享受、平台保障，不重复产品功能卖点。
- 如果底部拥挤，优先删除下层服务胶囊，不要删除主价格、赠品实物和赠品价值。
- 腰带下移时要靠近画面底边，但不要裁切价格数字和赠品。

### 2. 平台补贴白红价格区

适合：

- 清透主图
- 黑色产品
- 高质感家电
- 想避免画面太吵
- 平台补贴、价保、到手价

视觉关键词：

- 白底红字
- 红色描边
- 轻金色边
- 平台补贴标签
- 干净价格卡
- 红色角标
- 浅色底板

推荐提示词：

```text
底部使用白底红字平台补贴价格区，红色描边，轻金色边框，价格数字清楚醒目，搭配蓝色品牌条和浅色背景，整体清透不吵。
```

---

### 3. 黑金价格铭牌

适合：

- 高端款
- 金属质感产品
- 0 涂层、不锈钢、精钢厚釜
- 想做高级但仍有价格感

视觉关键词：

- 黑底金边
- 价格白金字
- 金属铭牌
- 少量光效
- 金色扫光
- 高端质感

推荐提示词：

```text
价格区使用黑金金属铭牌，黑底金边，价格白金字，金色扫光和轻微高光，整体高级克制，不使用大面积火焰。
```

## 二级备选：活动和玩法型

### 4. 优惠券价格卡

适合：

- 领券
- 咨询客服
- 返现
- 赠品
- 到手价由优惠组成

视觉关键词：

- 优惠券形状
- 撕纸边
- 虚线
- 领券按钮
- 白底红边
- 券后价

推荐提示词：

```text
价格区使用优惠券卡片，白底红边，券形虚线和撕纸边，突出券后到手价，适合领券、返现、赠品类活动。
```

---

### 5. 闪电直降条

适合：

- 秒杀
- 限时直降
- 立减
- 抢购

视觉关键词：

- 斜切红条
- 闪电图形
- 速度线
- 价格冲破边框
- 爆发光
- 倒计时感

注意：

- 不建议每张都用，容易显得很吵。
- 适合作为节点活动图或限时秒杀图。

---

## 右侧权益条规则

价格区右侧优先写权益，而不是重复卖点。

常用权益：

- 京仓直发
- 送货上门
- 价保 618
- 早买早享受
- 365 天只换不修
- 运费险
- 全国联保

示例：

```text
京仓直发 送货上门
价保618 早买早享受
```

视觉：

- 红粉权益背景
- 白粉胶囊条
- 小爱心图标
- 服务保障章
- 官方背书章

禁忌：

- 不要重复左侧小卖点。
- 不要把权益条做得比价格更抢。
- 不要让长文字挤压变形。

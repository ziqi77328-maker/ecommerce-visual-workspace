# E-价格视觉风格

> 用于解决价格区同质化问题。
> 价格区不要每张都默认“红金火焰底栏”，应根据产品调性、促销强度和画面风格选择。

---

## 使用原则

价格区首先要清楚，其次才是特效。

```text
强促销 -> 红金促销腰带
清透高光 -> 白红平台补贴区
高端金属 -> 黑金价格铭牌
领券返现 -> 优惠券价格卡
秒杀直降 -> 闪电直降条
```

注意：

- 价格数字必须是价格区最大元素。
- 右侧优先放权益，不要重复小卖点。
- 价格区不能遮挡产品主体和产品证据。
- 同一系列多张图要轮换价格视觉，避免同质化。
- 火焰不是必要视觉元素，只在强促销且不影响产品主体时使用。

---

## 一级常用：主图首发推荐

### 1. 红金促销腰带

适合：

- 618、双 11、平台大促
- 到手价
- 限时活动
- 强低价冲击

视觉关键词：

- 底部红金横条
- 火焰底托（可选）
- 金色描边
- 超大价格数字
- 白黄渐变数字
- 红金爆闪
- 速度线
- 流光

推荐结构：

```text
左侧活动标签
+
中间超大价格
+
右侧权益条
```

示例：

```text
活动到手价
¥799
京仓直发 送货上门 / 价保618 早买早享受
```

注意：

- 右侧放权益，不要重复“23L、0涂层、空气炸”等小卖点。
- 适合强促销，但不能每张都用。
- 火焰可以替换为流光、爆闪、描边、速度线或金属高光，不能把火焰当成默认必选项。

---

### 2. 平台补贴白红价格区

适合：

- 清透主图
- 黑色产品
- 高质感家电
- 想避免画面太吵
- 平台补贴、价保、到手价

视觉关键词：

- 白底红字
- 红色描边
- 轻金色边
- 平台补贴标签
- 干净价格卡
- 红色角标
- 浅色底板

推荐提示词：

```text
底部使用白底红字平台补贴价格区，红色描边，轻金色边框，价格数字清楚醒目，搭配蓝色品牌条和浅色背景，整体清透不吵。
```

---

### 3. 黑金价格铭牌

适合：

- 高端款
- 金属质感产品
- 0 涂层、不锈钢、精钢厚釜
- 想做高级但仍有价格感

视觉关键词：

- 黑底金边
- 价格白金字
- 金属铭牌
- 少量光效
- 金色扫光
- 高端质感

推荐提示词：

```text
价格区使用黑金金属铭牌，黑底金边，价格白金字，金色扫光和轻微高光，整体高级克制，不使用大面积火焰。
```

---

## 二级备选：活动和玩法型

### 4. 优惠券价格卡

适合：

- 领券
- 咨询客服
- 返现
- 赠品
- 到手价由优惠组成

视觉关键词：

- 优惠券形状
- 撕纸边
- 虚线
- 领券按钮
- 白底红边
- 券后价

推荐提示词：

```text
价格区使用优惠券卡片，白底红边，券形虚线和撕纸边，突出券后到手价，适合领券、返现、赠品类活动。
```

---

### 5. 闪电直降条

适合：

- 秒杀
- 限时直降
- 立减
- 抢购

视觉关键词：

- 斜切红条
- 闪电图形
- 速度线
- 价格冲破边框
- 爆发光
- 倒计时感

注意：

- 不建议每张都用，容易显得很吵。
- 适合作为节点活动图或限时秒杀图。

---

## 右侧权益条规则

价格区右侧优先写权益，而不是重复卖点。

常用权益：

- 京仓直发
- 送货上门
- 价保 618
- 早买早享受
- 365 天只换不修
- 运费险
- 全国联保

示例：

```text
京仓直发 送货上门
价保618 早买早享受
```

视觉：

- 红粉权益背景
- 白粉胶囊条
- 小爱心图标
- 服务保障章
- 官方背书章

禁忌：

- 不要重复左侧小卖点。
- 不要把权益条做得比价格更抢。
- 不要让长文字挤压变形。

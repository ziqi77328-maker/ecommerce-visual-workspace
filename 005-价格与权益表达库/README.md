# 005-价格与权益表达库

> 用于沉淀电商视觉中高转化价格区的表达逻辑。
>
> 重点研究：
>
> - 价格放在哪里
> - 如何与卖点配合
> - 如何突出优惠感
> - 如何提升点击率

---

## 高转化价格区原则

- 价格必须形成独立视觉锚点，不能只作为普通文字出现
- 活动价数字要大，建议占底部价格区高度的60%以上
- 价格区常用红金高饱和配色，适合京东促销氛围
- 数字可突破容器边界，制造冲击力
- 权益信息要和价格放在同一成交区域，减少用户二次寻找
- 赠品、返现、限时活动要图形化表达

## 电饭煲常用价格权益组合

- 618活动到手价 + 超大价格数字
- 晒单返现 + 返现金额放大
- 赠送全套配件 + 配件实物小图横排
- 真正低价 + 限时开抢口号
- 红金火焰底栏 + 流光 + 高光描边

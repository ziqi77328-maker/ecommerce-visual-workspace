# 005-价格与权益表达库

> 用于沉淀电商视觉中高转化价格区的表达逻辑。
>
> 重点研究：
>
> - 价格放在哪里
> - 如何与卖点配合
> - 如何突出优惠感
> - 如何提升点击率

---

## 高转化价格区原则

- 价格必须形成独立视觉锚点，不能只作为普通文字出现
- 活动价数字要大，建议占底部价格区高度的60%以上
- 价格区常用红金高饱和配色，适合京东促销氛围
- 数字可突破容器边界，制造冲击力
- 权益信息要和价格放在同一成交区域，减少用户二次寻找
- 赠品、返现、限时活动要图形化表达

## 电饭煲常用价格权益组合

- 活动到手价 + 超大价格数字
- 晒单返现 + 返现金额放大
- 赠送全套配件 + 配件实物小图横排
- 真正低价 + 限时促销口号
- 红金火焰底栏 + 流光 + 高光描边
- 底部红金火焰价格腰带 + 右侧黑金权益条

---

## 底部价格腰带结构

适合京东/天猫强促销主图，尤其是左文右品结构。

```text
左侧活动标签
+
中间超大价格数字
+
右侧权益承诺
+
红金火焰底托
```

示例：

```text
活动到手价
¥399
365天只换不修 | 价保30天
```

视觉关键词：

- 底部红金火焰价格腰带
- 超大价格数字
- 白黄渐变数字
- 金色描边
- 3D 阴影
- 火焰底托
- 红色爆闪
- 金色流光
- 速度线
- 黑金权益条

注意：

- 价格数字必须是底部最大视觉元素。
- 权益信息靠近价格，减少用户二次寻找。
- 价格腰带要横跨底部，不能只做小贴片。
- 火焰和流光不能遮挡产品主体。

## 标准成交模块

一张高转化主图的价格区建议包含：

```text
活动标签 + 到手价 + 立省/返现 + 赠品 + 限时口号 + 售后保障
```

示例：

```text
活动到手价 ¥169
晒单返现10元
赠配件全套
真正低价 限时促销
365天只换不修 / 京东速发 / 价保30天
```

---

## 视觉原则

- 价格数字必须是底部最大元素。
- 红金配色适合强促销，黑金配色适合高客单或高端款。
- 返现、赠品、补贴、PLUS权益要靠近价格，不要分散到画面边角。
- 赠品尽量用实物小图横排，不只写文字。
- 价格区可以有火焰、流光、描边，但不能遮挡产品主体。

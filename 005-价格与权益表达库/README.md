# 005-价格与权益表达库

> 用于沉淀电商视觉中高转化价格区的表达逻辑。
>
> 重点研究：
>
> - 价格放在哪里
> - 如何与卖点配合
> - 如何突出优惠感
> - 如何提升点击率

---

## 高转化价格区原则

- 价格必须形成独立视觉锚点，不能只作为普通文字出现
- 活动价数字要大，建议占底部价格区高度的60%以上
- 价格区常用红金高饱和配色，适合京东促销氛围
- 数字可突破容器边界，制造冲击力
- 权益信息要和价格放在同一成交区域，减少用户二次寻找
- 赠品、返现、限时活动要图形化表达
- 火焰不是价格区的必要元素，可根据产品调性、促销强度和画面干净程度改用流光、描边、爆闪、金属铭牌、优惠券卡或平台补贴卡

## 价格权益真实性与去重复规则

- 没有真实价格、返现、赠品、补贴或售后承诺时，只写占位符，不编造金额和承诺。
- 底部权益区优先写成交权益，不重复左侧产品功能卖点。
- 可写成交权益：京仓直发、送货上门、价保、只换不修、全国联保、正品保障、运费险、限时发货。
- 产品参数和功能卖点应放在卖点区或产品证据区，不放在价格权益区反复出现。
- 主图提示词里必须明确“活动价 + 至少 2 个成交权益”，否则容易生成只有产品和卖点、没有成交触发的画面。

## 电饭煲常用价格权益组合

- 活动到手价 + 超大价格数字
- 晒单返现 + 返现金额放大
- 赠送全套配件 + 配件实物小图横排
- 真正低价 + 限时促销口号
- 红金促销底栏 + 流光 + 高光描边
- 底部促销价格腰带 + 右侧黑金权益条

## 底部价格腰带结构

适合京东/天猫强促销主图，尤其是左文右品结构。

```text
左侧活动标签
+
中间超大价格数字
+
右侧权益承诺
```

示例：

```text
活动到手价
¥399
365天只换不修 | 价保30天
```

视觉关键词：

- 底部红金促销价格腰带
- 超大价格数字
- 白黄渐变数字
- 金色描边
- 3D 阴影
- 火焰底托（可选）
- 红色爆闪
- 金色流光
- 速度线
- 黑金权益条

注意：

- 价格数字必须是底部最大视觉元素。
- 权益信息靠近价格，减少用户二次寻找。
- 价格腰带要横跨底部，不能只做小贴片。
- 主价格腰带高度控制在画面高度 14%-18%。
- 如果有底部服务条，总成交区高度不能超过 20%-22%。
- 价格区贴近底边，不遮挡产品主体、内胆证据、右侧贴片和左侧卖点区。
- 如果价格腰带超过画面高度 22%，视为不合格，需要压缩。
- 火焰不是必选项；如使用火焰、流光或爆闪，不能遮挡产品主体和产品证据。

## 价格腰带标准规格

1:1 主图中：

- 主价格腰带高度：画面高度的 14%-18%。
- 含底部服务条的总成交区高度：不超过 20%-22%。
- 服务权益条可以放在腰带下层，但高度建议 4%-5%。
- 价格数字是腰带内最大元素，但不能让腰带变成画面主视觉。
- 价格区贴近底边，不遮挡产品主体、内胆证据、右侧贴片和左侧卖点区。
- 如果价格腰带超过画面高度 22%，视为不合格，需要压缩重做。

标准提示词：

```text
底部价格腰带高度控制在画面 14%-18%，如果有下层服务权益条，总高度不超过 22%。价格区贴近底边，不遮挡产品主体、内胆证据、右侧贴片和左侧卖点区。
```

## 右侧权益条表达

底部价格腰带右侧优先放权益，不要重复左侧小卖点。

适合：

- 京东仓配
- 上门配送
- 价保
- 早买早享受
- 只换不修
- 运费险

示例：

```text
京仓直发 送货上门
价保618 早买早享受
```

视觉关键词：

- 红粉权益背景
- 白粉胶囊条
- 大号浅黄色权益文字
- 红粉强调字
- 小爱心图标
- 服务保障模块

注意：

- 不要把产品功能卖点重复放到底部右侧。
- 权益条要和价格靠近，形成成交闭环。
- 文字较长时缩小字号或扩大胶囊条，避免挤压。
- 价格腰带主权益和下方小贴片不能重复表达同一件事。
- 每个权益只出现一次，主权益负责最大成交刺激，小贴片只补充不同服务或限制条件。

重复错误：

```text
主权益：今日前10名下单送价值209元电磁炉
小贴片：限量赠品 / 前10名专享 / 下单即赠 / 赠完即止
```

推荐方案：

```text
方案 A：
主权益：送价值209元电磁炉
小贴片：前10名专享 / 随单发出 / 咨询客服 / 库存有限

方案 B：
主权益：今日前10名下单专享
小贴片：赠电磁炉 / 价保30天 / 京仓速发 / 正品保障
```

## 赠品与价格腰带关系

赠品不是普通装饰，应当成为价格区的一部分，证明“这个价格还额外送什么”。

高转化关系：

```text
超大价格
+
赠品实物压在腰带上沿
+
赠品价值文案紧贴实物
+
价保/早买早享受放在下层服务条
```

表达规则：

- 赠品实物优先放在价格数字与赠品文案之间，形成视觉桥梁。
- 赠品可以从腰带上沿向上突破，但不能遮挡主价格数字。
- “赠”角标要贴近赠品实物，不要单独漂浮。
- 赠品价值金额要大于普通权益字，但小于主价格数字。
- 晒单返现、立减、补贴可以放在腰带上方，作为额外刺激，不抢主价格。
- 下层服务胶囊只放价保、配送、售后、早买早享受等成交保障。

提示词：

```text
赠品实物小图贴着底部价格腰带上沿摆放，从腰带中间向上突破，旁边带红色“赠”角标；赠品价值文案紧贴赠品右侧，金额用亮黄色放大。主价格仍是最大元素，赠品和返现作为成交加码，不遮挡价格数字。
```

## 标准成交模块

一张高转化主图的价格区建议包含：

```text
活动标签 + 到手价 + 立省/返现 + 赠品 + 限时口号 + 售后保障
```

示例：

```text
活动到手价 ¥169
晒单返现10元
赠配件全套
真正低价 限时促销
365天只换不修 / 京东速发 / 价保30天
```

## 视觉原则

- 价格数字必须是底部最大元素。
- 红金配色适合强促销，黑金配色适合高客单或高端款。
- 返现、赠品、补贴、PLUS权益要靠近价格，不要分散到画面边角。
- 赠品尽量用实物小图横排，不只写文字。
- 价格区可以有流光、描边、爆闪、火焰等促销元素，但火焰不是必要元素，所有特效都不能遮挡产品主体和产品证据。
